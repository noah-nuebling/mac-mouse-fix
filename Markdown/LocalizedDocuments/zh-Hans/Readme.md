<a name="macos-compatibility"></a>
## macOS 兼容性

最新版本的 Mac Mouse Fix 专为 **macOS 11 Big Sur** 或更高版本设计。
  
如果你使用的是 macOS **10.15 Catalina**、macOS **10.14 Mojave** 或 macOS **10.13 High Sierra**，可以使用 [Mac Mouse Fix 2 的最新版本](https://redirect.macmousefix.com/?locale=zh-Hans&target=mmf2-latest)。Mac Mouse Fix 3.0.0 及更高版本可能仍能在这些系统上运行，但会出现视觉瑕疵，且部分功能可能无法正常使用。
    
如果你使用的是 macOS **10.12 Sierra** 或 **10.11 El Capitan**，可以使用 Mac Mouse Fix [2.2.0](https://github.com/noah-nuebling/mac-mouse-fix/releases/tag/2.2.0) 或更低版本。

<a name="tips"></a> 
## 提示

- 简单的一点一拖，即可轻松管理窗口

  [Swish](https://highlyopinionated.co/swish/) 是我在 macOS 上最喜欢的窗口管理方式。通过在触控板上简单一划，它就能让窗口快速定位，占据半屏、四分之一屏或全屏。
  
  Swish 专为触控板手势设计，但配合 Mac Mouse Fix，你在任何第三方鼠标上都能使用它！只需在 Mac Mouse Fix 中将任意按键的“点击并拖拽（Click and Drag）”动作设为“滚动与导航（Scroll & Navigate）”，即可通过简单的点按拖动来实现窗口吸附。
  
  任何你在 Apple 触控板上通过双指滑动实现的操作，在 Mac Mouse Fix 的“滚动与导航”功能下都能获得同样出色的体验。

- **直接通过鼠标控制屏幕亮度、音量大小或媒体播放**

  Mac Mouse Fix 让你能直接通过鼠标使用**键盘上的任何按键** —— 
  甚至包括 Apple 键盘特有的功能键，从而实现对屏幕亮度、音量大小、媒体播放等功能的直接控制。
  
  如果你的手边没有 Apple 键盘，只需**按住 Option (⌥) 键**，即可选择这些特殊的 Apple 功能键。

<a name="questions"></a> 
## 常见问题

- Mac Mouse Fix 原生支持 Apple Silicon 吗？
    
  是的，Mac Mouse Fix 在 Apple Silicon 芯片上 100% 原生运行。

- 为什么点击鼠标时感觉有延迟？

  当你点击时，Mac Mouse Fix 可能会等待后续操作，以判断你是否要进行“连按”。<br>
  若要消除某个按键的点击延迟，请删除该按键下所有的“连按”操作指令。

- ** 是否支持我的鼠标？**

  大概率可以。如果你想确切了解兼容性，最好的办法是下载 Mac Mouse Fix 并亲自尝试一下。
  
  Mac Mouse Fix 与绝大多数鼠标都能完美配合。然而，对于某些设计上必须配合特定驱动软件（如 Logitech Options）使用的鼠标，Mac Mouse Fix 目前可能无法识别其所有按键。
  
  这是因为这些鼠标使用的是特殊的私有协议与电脑通信，而不是标准的 USB 协议。<br>
  我非常希望在未来能实现对这些鼠标的完全兼容，但这涉及巨大的工作量，短期内可能无法实现。

- **Why do the 'Scrolling' and 'Buttons' items in the Menu Bar get re-enabled?**
    
  The Menu Bar buttons are meant to quickly, and temporarily turn off certain features while you're working in incompatible apps. 
  This is intended as a stop-gap-measure while there's no 'App-Specific Settings' feature, yet. <br>
  Curently the UI would break if these are enabled, so when you open the Buttons tab or Scrolling tab, they get toggled and stuff.
  To edit the settings in a permanent way, you can edit the settings in 'Buttons' and 'Scrolling' Tab. 
  
  To completely turn off all effects of Mac Mouse Fix on your buttons, delete all the actions on the 'Buttons' tab. <br>
  To completely turn off all effects of Mac Mouse Fix on your scroll wheel, configure the 'Scrolling' tab like this:

## 如何参与贡献

你可以通过多种方式为本项目提供帮助。<br>
请查看[致谢名单](Acknowledgements.md)，了解更多关于已有贡献者的信息！

- **Giving feedback**
    
  你可以通过 [反馈助手 (Feedback Assistant)](https://noah-nuebling.github.io/mac-mouse-fix-feedback-assistant/?type=bug-report) 分享你的**想法**、**问题** 以及**建议**，以此来帮助我们改进。

- **捐赠支持**
  
  我希望能通过 Mac Mouse Fix 实现经济上的自给自足，这样我就能持续投入精力来改进和开发这款应用。如果你愿意提供帮助，可以通过以下方式：
  
  1. **购买 Mac Mouse Fix**：你可以点击应用内的按钮，或点击[这里](https://noahnuebling.gumroad.com/l/mmfinappusd)直接购买。
  2. **通过 PayPal [打赏我](https://www.paypal.com/cgi-bin/webscr?cmd=_s-xclick&hosted_button_id=ARSTVR6KFB524&source=url&lc=en_US)**：虽然这不会带来很多收入，但每收到一份捐赠都会让我觉得很暖心，也是一种极大的激励。
  3. **在 GitHub 上 [资助我](https://github.com/sponsors/noah-nuebling)**：每月的定期资助是支持本项目并帮助我获得稳定收入的绝佳方式。

- **参与翻译贡献**
  
  Mac Mouse Fix 目前支持英语、德语，以及[致谢名单](Acknowledgements.md)中所列出的各项语言。
  
  如果你想**帮助翻译本项目**，请参阅[翻译指南](https://redirect.macmousefix.com/?target=mmf-localization-contribution)。<br>
  如果你想**报告翻译缺失或翻译不佳的问题**，这也将对我们有很大帮助。报告此类问题的最佳方式是在[翻译指南](https://redirect.macmousefix.com/?target=mmf-localization-contribution)下方留言。

- **参与代码贡献**

  如果你愿意为本项目贡献代码，那真是太棒了！我非常欢迎任何形式的 [Pull Request (PR)](https://github.com/noah-nuebling/mac-mouse-fix/pulls)。
  
  不过，我可能无法接受所有的 PR。为了确保你的辛勤工作不被浪费，建议你可以先提交一个初步的 PR——只需在其中“描述”你想要进行的更改，而无需（或只需包含少量）实际代码。这样我可以先为你提供反馈，并确认我是否会以这种方式采纳你所建议的改动。
