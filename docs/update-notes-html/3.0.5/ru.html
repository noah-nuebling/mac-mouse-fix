Mac Mouse Fix 3.0.5 исправляет несколько ошибок, улучшает производительность и добавляет немного полировки в приложение.
Также совместимо с macOS 26 Tahoe.

Улучшенная симуляция прокрутки трекпада

  • Система прокрутки теперь может симулировать двойное касание трекпада двумя пальцами, чтобы приложения останавливали прокрутку.
    • Это исправляет проблему при запуске приложений для iPhone или iPad, когда прокрутка часто продолжалась после того, как пользователь решал остановиться.
  • Исправлена непоследовательная симуляция отрыва пальцев от трекпада.
    • Это могло вызывать неоптимальное поведение в некоторых ситуациях.

Совместимость с macOS 26 Tahoe

При работе с бета-версией macOS 26 Tahoe приложение теперь работает, и большая часть интерфейса функционирует корректно.

Повышение производительности

Улучшена производительность жеста «Нажать и перетащить» для «Прокрутки и навигации».
В моих тестах использование процессора снизилось примерно на 50%!

Предыстория

Во время жеста «Прокрутка и навигация» Mac Mouse Fix рисует фальшивый курсор мыши в прозрачном окне, блокируя настоящий курсор мыши на месте. Это гарантирует, что ты можешь продолжать прокручивать элемент интерфейса, с которого начал прокрутку, независимо от того, как далеко ты переместишь мышь.

Улучшенная производительность была достигнута путём отключения стандартной обработки событий macOS в этом прозрачном окне, которая всё равно не использовалась.

Исправления ошибок

  • Теперь игнорируются события прокрутки от графических планшетов Wacom.
    • Раньше Mac Mouse Fix вызывал хаотичную прокрутку на планшетах Wacom, как сообщил @frenchie1980 в GitHub Issue #1233. (Спасибо!)
  • Исправлена ошибка, при которой код Swift Concurrency, введённый как часть новой системы лицензирования в Mac Mouse Fix 3.0.4, не выполнялся в правильном потоке.
    • Это вызывало сбои в macOS Tahoe, а также, вероятно, вызывало другие спорадические ошибки, связанные с лицензированием.
  • Улучшена надёжность кода, который декодирует офлайн-лицензии.
    • Это обходит проблему в API Apple, из-за которой проверка офлайн-лицензий всегда завершалась неудачей на моём Intel Mac Mini. Предполагаю, что это происходило на всех Mac с Intel, и это была причина, по которой ошибка «Бесплатные дни закончились» (которая уже была устранена в 3.0.4) всё ещё возникала у некоторых людей, как сообщил @toni20k5267 в GitHub Issue #1356. (Спасибо!)
      • Если ты столкнулся с ошибкой «Бесплатные дни закончились», мне очень жаль! Ты можешь получить возврат средств здесь.

Улучшения UX

  • Отключены диалоговые окна, которые предоставляли пошаговые решения для ошибок macOS, мешавших пользователям включить Mac Mouse Fix.
    • Эти проблемы возникали только в macOS 13 Ventura и 14 Sonoma. Теперь эти диалоговые окна появляются только в тех версиях macOS, где они актуальны.
    • Диалоговые окна также немного сложнее вызвать — раньше они иногда появлялись в ситуациях, где не были особо полезны.
  • Добавлена ссылка «Активировать лицензию» прямо в уведомлении «Бесплатные дни закончились».
    • Это делает активацию лицензии Mac Mouse Fix ещё более удобной!

Визуальные улучшения

  • Немного улучшен внешний вид окна «Обновление ПО». Теперь оно лучше сочетается с macOS 26 Tahoe.
    • Это было сделано путём настройки стандартного вида фреймворка «Sparkle 1.27.3», который Mac Mouse Fix использует для обработки обновлений.
  • Исправлена проблема, при которой текст внизу вкладки «О программе» иногда обрезался на китайском языке, за счёт небольшого расширения окна.
  • Исправлено небольшое смещение текста внизу вкладки «О программе» относительно центра.
  • Исправлена ошибка, из-за которой пространство под опцией «Сочетание клавиш…» на вкладке «Кнопки» было слишком маленьким.

Внутренние изменения

  • Удалена зависимость от фреймворка «SnapKit».
    • Это немного уменьшает размер приложения с 19,8 до 19,5 МБ.
  • Различные другие небольшие улучшения в кодовой базе.
