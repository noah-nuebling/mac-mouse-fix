ℹ️ แปลโดย AI

บันทึกการเปิดตัวนี้ได้รับการแปลโดย Claude AI อาจมีข้อผิดพลาดบางประการ
สำหรับเวอร์ชันภาษาอังกฤษต้นฉบับ คลิกที่นี่

Mac Mouse Fix 3.0.5 แก้ไขบั๊กหลายจุด ปรับปรุงประสิทธิภาพ และเพิ่มความสวยงามให้กับแอปอีกเล็กน้อย
นอกจากนี้ยังรองรับ macOS 26 Tahoe อีกด้วย

ปรับปรุงการจำลองการเลื่อนด้วยแทร็คแพด

  • ระบบการเลื่อนสามารถจำลองการแตะสองนิ้วบนแทร็คแพดเพื่อให้แอปพลิเคชันหยุดการเลื่อนได้แล้ว
    • แก้ไขปัญหาเมื่อใช้งานแอป iPhone หรือ iPad ที่การเลื่อนมักจะดำเนินต่อไปแม้ผู้ใช้จะเลือกหยุดแล้ว
  • แก้ไขการจำลองการยกนิ้วออกจากแทร็คแพดที่ไม่สม่ำเสมอ
    • อาจทำให้เกิดพฤติกรรมที่ไม่เหมาะสมในบางสถานการณ์

รองรับ macOS 26 Tahoe

เมื่อใช้งาน macOS 26 Tahoe Beta แอปสามารถใช้งานได้แล้ว และ UI ส่วนใหญ่ทำงานได้อย่างถูกต้อง

เพิ่มประสิทธิภาพ

ปรับปรุงประสิทธิภาพของท่าทางคลิกและลากเพื่อ “เลื่อนและนำทาง”
จากการทดสอบของผม การใช้ CPU ลดลงประมาณ 50%!

ข้อมูลเบื้องหลัง

ระหว่างท่าทาง “เลื่อนและนำทาง” Mac Mouse Fix จะวาดเคอร์เซอร์เมาส์ปลอมในหน้าต่างโปร่งใส พร้อมกับล็อกเคอร์เซอร์เมาส์จริงไว้ที่เดิม วิธีนี้ช่วยให้คุณสามารถเลื่อนองค์ประกอบ UI ที่คุณเริ่มเลื่อนต่อไปได้ ไม่ว่าคุณจะเลื่อนเมาส์ไปไกลแค่ไหน

การปรับปรุงประสิทธิภาพทำได้โดยการปิดการจัดการอีเวนต์เริ่มต้นของ macOS บนหน้าต่างโปร่งใสนี้ ซึ่งไม่ได้ใช้งานอยู่แล้ว

แก้ไขบั๊ก

  • ตอนนี้จะเพิกเฉยต่ออีเวนต์การเลื่อนจากแท็บเล็ตวาด Wacom
    • ก่อนหน้านี้ Mac Mouse Fix ทำให้เกิดการเลื่อนที่ผิดปกติบนแท็บเล็ต Wacom ตามที่ @frenchie1980 รายงานใน GitHub Issue #1233 (ขอบคุณ!)
  • แก้ไขบั๊กที่โค้ด Swift Concurrency ซึ่งถูกเพิ่มเข้ามาเป็นส่วนหนึ่งของระบบไลเซนส์ใหม่ใน Mac Mouse Fix 3.0.4 ไม่ทำงานบนเธรดที่ถูกต้อง
    • ทำให้เกิดการแครชบน macOS Tahoe และน่าจะเป็นสาเหตุของบั๊กเกี่ยวกับไลเซนส์ที่เกิดขึ้นเป็นครั้งคราวด้วย
  • ปรับปรุงความแข็งแกร่งของโค้ดที่ถอดรหัสไลเซนส์ออฟไลน์
    • แก้ไขปัญหาใน API ของ Apple ที่ทำให้การตรวจสอบไลเซนส์ออฟไลน์ล้มเหลวเสมอบน Intel Mac Mini ของผม ผมคิดว่าปัญหานี้เกิดขึ้นบน Intel Mac ทุกเครื่อง และน่าจะเป็นเหตุผลที่บั๊ก “Free days are over” (ที่แก้ไขไปแล้วใน 3.0.4) ยังคงเกิดขึ้นกับบางคน ตามที่ @toni20k5267 รายงานใน GitHub Issue #1356 (ขอบคุณ!)
      • หากคุณประสบปัญหาบั๊ก “Free days are over” ผมขอโทษด้วย! คุณสามารถขอคืนเงินได้ที่นี่

ปรับปรุง UX

  • ปิดการใช้งานไดอะล็อกที่ให้วิธีแก้ปัญหาทีละขั้นตอนสำหรับบั๊กของ macOS ที่ทำให้ผู้ใช้ไม่สามารถเปิดใช้งาน Mac Mouse Fix ได้
    • ปัญหาเหล่านี้เกิดขึ้นเฉพาะบน macOS 13 Ventura และ 14 Sonoma ตอนนี้ไดอะล็อกเหล่านี้จะปรากฏเฉพาะบน macOS เวอร์ชันที่เกี่ยวข้องเท่านั้น
    • ไดอะล็อกยังถูกทริกเกอร์ได้ยากขึ้นเล็กน้อย – ก่อนหน้านี้บางครั้งจะแสดงขึ้นในสถานการณ์ที่ไม่ค่อยมีประโยชน์
  • เพิ่มลิงก์ “เปิดใช้งานไลเซนส์” โดยตรงบนการแจ้งเตือน “Free days are over”
    • ทำให้การเปิดใช้งานไลเซนส์ Mac Mouse Fix ง่ายและสะดวกยิ่งขึ้น!

ปรับปรุงภาพลักษณ์

  • ปรับปรุงรูปลักษณ์ของหน้าต่าง “Software Update” เล็กน้อย ตอนนี้เข้ากับ macOS 26 Tahoe ได้ดีขึ้น
    • ทำได้โดยการปรับแต่งรูปลักษณ์เริ่มต้นของเฟรมเวิร์ก “Sparkle 1.27.3” ที่ Mac Mouse Fix ใช้จัดการอัปเดต
  • แก้ไขปัญหาที่ข้อความด้านล่างของแท็บ About ถูกตัดบางครั้งในภาษาจีน โดยทำให้หน้าต่างกว้างขึ้นเล็กน้อย
  • แก้ไขข้อความด้านล่างของแท็บ About ที่เยื้องไม่ตรงกลางเล็กน้อย
  • แก้ไขบั๊กที่ทำให้พื้นที่ใต้ตัวเลือก “Keyboard Shortcut…” บนแท็บ Buttons เล็กเกินไป

การเปลี่ยนแปลงภายใน

  • ลบการพึ่งพาเฟรมเวิร์ก “SnapKit”
    • ทำให้ขนาดของแอปลดลงเล็กน้อยจาก 19.8 เป็น 19.5 MB
  • ปรับปรุงเล็กน้อยอื่นๆ ในโค้ดเบส
