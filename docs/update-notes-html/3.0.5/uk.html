Mac Mouse Fix 3.0.5 виправляє кілька помилок, покращує продуктивність і додає трохи шліфування до програми.
Також сумісний з macOS 26 Tahoe.

Покращена симуляція прокручування трекпадом

  • Система прокручування тепер може симулювати подвійне торкання двома пальцями на трекпаді, щоб зупинити прокручування в програмах.
    • Це виправляє проблему при запуску програм для iPhone або iPad, коли прокручування часто продовжувалося після того, як користувач вирішив зупинитися.
  • Виправлено непослідовну симуляцію відривання пальців від трекпада.
    • Це могло спричиняти неоптимальну поведінку в деяких ситуаціях.

Сумісність з macOS 26 Tahoe

При роботі з бета-версією macOS 26 Tahoe програма тепер працює, і більшість інтерфейсу функціонує коректно.

Підвищення продуктивності

Покращено продуктивність жесту «Клацання та перетягування» для «Прокручування та навігації».
У моїх тестах використання процесора зменшилося приблизно на 50%!

Передісторія

Під час жесту «Прокручування та навігації» Mac Mouse Fix малює фальшивий курсор миші в прозорому вікні, одночасно фіксуючи справжній курсор миші на місці. Це гарантує, що ти можеш продовжувати прокручувати елемент інтерфейсу, з якого почав, незалежно від того, як далеко рухаєш мишу.

Покращення продуктивності було досягнуто шляхом вимкнення стандартної обробки подій macOS для цього прозорого вікна, яка все одно не використовувалася.

Виправлення помилок

  • Тепер ігноруються події прокручування від графічних планшетів Wacom.
    • Раніше Mac Mouse Fix спричиняв хаотичне прокручування на планшетах Wacom, як повідомив @frenchie1980 у GitHub Issue #1233. (Дякую!)
  • Виправлено помилку, коли код Swift Concurrency, який було введено як частину нової системи ліцензування в Mac Mouse Fix 3.0.4, не виконувався в правильному потоці.
    • Це спричиняло збої на macOS Tahoe, а також ймовірно викликало інші спорадичні помилки, пов’язані з ліцензуванням.
  • Покращено надійність коду, який декодує офлайн-ліцензії.
    • Це обходить проблему в API Apple, яка спричиняла постійну невдачу перевірки офлайн-ліцензій на моєму Intel Mac Mini. Я припускаю, що це траплялося на всіх Mac з Intel, і що це була причина, чому помилка «Безкоштовні дні закінчилися» (яку вже було вирішено в 3.0.4) все ще виникала у деяких людей, як повідомив @toni20k5267 у GitHub Issue #1356. (Дякую!)
      • Якщо ти зіткнувся з помилкою «Безкоштовні дні закінчилися», мені дуже шкода! Ти можеш отримати повернення коштів тут.

Покращення UX

  • Вимкнено діалоги, які надавали покрокові рішення для помилок macOS, що заважали користувачам увімкнути Mac Mouse Fix.
    • Ці проблеми виникали лише на macOS 13 Ventura та 14 Sonoma. Тепер ці діалоги з’являються лише на тих версіях macOS, де вони актуальні.
    • Діалоги також трохи складніше викликати – раніше вони іноді з’являлися в ситуаціях, де не були дуже корисними.
  • Додано посилання «Активувати ліцензію» безпосередньо в сповіщенні «Безкоштовні дні закінчилися».
    • Це робить активацію ліцензії Mac Mouse Fix ще більш безпроблемною!

Візуальні покращення

  • Трохи покращено вигляд вікна «Оновлення програмного забезпечення». Тепер воно краще поєднується з macOS 26 Tahoe.
    • Це було зроблено шляхом налаштування стандартного вигляду фреймворку «Sparkle 1.27.3», який Mac Mouse Fix використовує для обробки оновлень.
  • Виправлено проблему, коли текст внизу вкладки «Про програму» іноді обрізався китайською мовою, зробивши вікно трохи ширшим.
  • Виправлено текст внизу вкладки «Про програму», який був трохи не по центру.
  • Виправлено помилку, яка спричиняла занадто малий простір під опцією «Комбінація клавіш…» на вкладці «Кнопки».

Зміни під капотом

  • Видалено залежність від фреймворку «SnapKit».
    • Це трохи зменшує розмір програми з 19,8 до 19,5 МБ.
  • Різні інші дрібні покращення в кодовій базі.
