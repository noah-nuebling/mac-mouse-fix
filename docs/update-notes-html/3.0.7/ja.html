Mac Mouse Fix 3.0.7 では、いくつかの重要なバグに対処しました。

バグ修正

  • 古いmacOSバージョン(macOS 10.15 CatalinaおよびmacOS 11 Big Sur)でアプリが再び動作するようになりました
    • Mac Mouse Fix 3.0.6では、これらのmacOSバージョンで有効化できませんでした。これは、Mac Mouse Fix 3.0.6で導入された改良版の「戻る」および「進む」機能が、利用できないmacOSシステムAPIを使用しようとしたためです。
  • 「戻る」および「進む」機能の問題を修正しました
    • Mac Mouse Fix 3.0.6で導入された改良版の「戻る」および「進む」機能は、使用中のアプリで戻る・進むために模擬するキー入力をmacOSに問い合わせる際、常に「メインスレッド」を使用するようになりました。
      これにより、一部の状況でクラッシュや不安定な動作を防ぐことができます。
  • 設定がランダムにリセットされるバグの修正を試みました(GitHub Issuesを参照)
    • Mac Mouse Fixの設定ファイルを読み込むコードをより堅牢になるように書き直しました。まれにmacOSファイルシステムエラーが発生した際、古いコードは設定ファイルが破損していると誤って判断し、デフォルトにリセットすることがありました。
  • スクロールが動作しなくなるバグの発生確率を低減しました
    • このバグは、より深い変更なしには完全に解決できず、そうした変更は他の問題を引き起こす可能性があります。
      しかし、当面の間、スクロールシステムで「デッドロック」が発生する可能性のある時間枠を短縮し、少なくともこのバグに遭遇する確率を下げました。これにより、スクロールもわずかに効率的になりました。
    • このバグは、前回のリリース3.0.6で対処された「スクロールが断続的に動作しなくなる」バグと似た症状ですが、根本的な原因は異なると考えています。
    • (診断を提供してくれたJoonasに感謝します!)
