Mac Mouse Fix 3.0.7 виправляє кілька важливих помилок.

Виправлення помилок

  • Програма знову працює на старіших версіях macOS (macOS 10.15 Catalina та macOS 11 Big Sur)
    • Mac Mouse Fix 3.0.6 не вдавалося увімкнути на цих версіях macOS, оскільки покращена функція «Назад» і «Вперед», представлена в Mac Mouse Fix 3.0.6, намагалася використовувати системні API macOS, які були недоступні.
  • Виправлено проблеми з функцією «Назад» і «Вперед»
    • Покращена функція «Назад» і «Вперед», представлена в Mac Mouse Fix 3.0.6, тепер завжди використовуватиме «головний потік» для запиту до macOS про те, які натискання клавіш симулювати для переходу назад і вперед у програмі, яку ви використовуєте.
      Це може запобігти збоям і ненадійній поведінці в деяких ситуаціях.
  • Спроба виправити помилку, через яку налаштування випадково скидалися (Див. ці GitHub Issues)
    • Я переписав код, який завантажує файл конфігурації для Mac Mouse Fix, щоб зробити його більш надійним. Коли траплялися рідкісні помилки файлової системи macOS, старий код іноді міг помилково вважати, що файл конфігурації пошкоджений, і скидати його до стандартних налаштувань.
  • Зменшено ймовірність помилки, через яку прокручування переставало працювати
    • Цю помилку неможливо повністю вирішити без глибших змін, які, ймовірно, спричинили б інші проблеми.
      Однак наразі я зменшив часове вікно, коли може статися «взаємне блокування» в системі прокручування, що принаймні має знизити ймовірність виникнення цієї помилки. Це також робить прокручування трохи ефективнішим.
    • Ця помилка має схожі симптоми – але, на мою думку, іншу основну причину – з помилкою «Прокручування періодично переставало працювати», яку було виправлено в попередньому випуску 3.0.6.
    • (Дякую Joonas за діагностику!)

Дякуємо всім за повідомлення про помилки!
