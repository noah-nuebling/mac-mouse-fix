ℹ️ แปลโดย AI

บันทึกการเปิดตัวนี้ได้รับการแปลโดย Claude AI อาจมีข้อผิดพลาดบางประการ
สำหรับเวอร์ชันภาษาอังกฤษต้นฉบับ คลิกที่นี่

Mac Mouse Fix 3.0.7 แก้ไขบั๊กสำคัญหลายประการ

การแก้ไขบั๊ก

  • แอปทำงานได้อีกครั้งบนmacOS เวอร์ชันเก่า (macOS 10.15 Catalina และ macOS 11 Big Sur)
    • Mac Mouse Fix 3.0.6 ไม่สามารถเปิดใช้งานได้บน macOS เวอร์ชันเหล่านั้น เนื่องจากฟีเจอร์ ‘ย้อนกลับ’ และ ‘ไปข้างหน้า’ ที่ปรับปรุงใหม่ใน Mac Mouse Fix 3.0.6 พยายามใช้ macOS system APIs ที่ยังไม่มีให้ใช้งาน
  • แก้ไขปัญหาเกี่ยวกับฟีเจอร์ ‘ย้อนกลับ’ และ ‘ไปข้างหน้า’
    • ฟีเจอร์ ‘ย้อนกลับ’ และ ‘ไปข้างหน้า’ ที่ปรับปรุงใหม่ใน Mac Mouse Fix 3.0.6 จะใช้ ‘main thread’ เสมอในการถาม macOS เกี่ยวกับการกดปุ่มใดที่จะจำลองเพื่อย้อนกลับและไปข้างหน้าในแอปที่คุณกำลังใช้งาน
      สิ่งนี้สามารถป้องกันการขัดข้องและพฤติกรรมที่ไม่น่าเชื่อถือในบางสถานการณ์
  • พยายามแก้ไขบั๊กที่การตั้งค่าถูกรีเซ็ตแบบสุ่ม (ดู GitHub Issues เหล่านี้)
    • ฉันเขียนโค้ดที่โหลดไฟล์ config สำหรับ Mac Mouse Fix ใหม่เพื่อให้แข็งแกร่งขึ้น เมื่อเกิดข้อผิดพลาดของระบบไฟล์ macOS ที่หายาก โค้ดเก่าอาจคิดผิดว่าไฟล์ config เสียหายและรีเซ็ตเป็นค่าเริ่มต้น
  • ลดโอกาสของบั๊กที่การเลื่อนหยุดทำงาน
    • บั๊กนี้ไม่สามารถแก้ไขได้อย่างสมบูรณ์หากไม่มีการเปลี่ยนแปลงที่ลึกกว่านี้ ซึ่งอาจทำให้เกิดปัญหาอื่นๆ
      อย่างไรก็ตาม ในตอนนี้ ฉันได้ลดช่วงเวลาที่ ‘deadlock’ สามารถเกิดขึ้นในระบบการเลื่อน ซึ่งควรจะช่วยลดโอกาสในการพบบั๊กนี้อย่างน้อย นอกจากนี้ยังทำให้การเลื่อนมีประสิทธิภาพมากขึ้นเล็กน้อย
    • บั๊กนี้มีอาการคล้ายกัน – แต่ฉันคิดว่ามีสาเหตุพื้นฐานที่แตกต่างกัน – กับบั๊ก ‘Scroll Stops Working Intermittently’ ที่ได้รับการแก้ไขในรุ่นก่อนหน้า 3.0.6
    • (ขอบคุณ Joonas สำหรับการวินิจฉัย!)

ขอบคุณทุกคนที่รายงานบั๊ก!
