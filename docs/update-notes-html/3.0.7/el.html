Το Mac Mouse Fix 3.0.7 αντιμετωπίζει αρκετά σημαντικά σφάλματα.

Διορθώσεις Σφαλμάτων

  • Η εφαρμογή λειτουργεί ξανά σε παλαιότερες εκδόσεις macOS (macOS 10.15 Catalina και macOS 11 Big Sur)
    • Το Mac Mouse Fix 3.0.6 δεν μπορούσε να ενεργοποιηθεί σε αυτές τις εκδόσεις macOS επειδή η βελτιωμένη λειτουργία ‘Πίσω’ και ‘Μπροστά’ που εισήχθη στο Mac Mouse Fix 3.0.6 προσπαθούσε να χρησιμοποιήσει APIs συστήματος macOS που δεν ήταν διαθέσιμα.
  • Διορθώθηκαν προβλήματα με τη λειτουργία ‘Πίσω’ και ‘Μπροστά’
    • Η βελτιωμένη λειτουργία ‘Πίσω’ και ‘Μπροστά’ που εισήχθη στο Mac Mouse Fix 3.0.6 θα χρησιμοποιεί πλέον πάντα το ‘main thread’ για να ρωτήσει το macOS ποια πλήκτρα να προσομοιώσει για να πας πίσω και μπροστά στην εφαρμογή που χρησιμοποιείς.
      Αυτό μπορεί να αποτρέψει crashes και αναξιόπιστη συμπεριφορά σε ορισμένες περιπτώσεις.
  • Προσπάθεια διόρθωσης σφάλματος όπου οι ρυθμίσεις επαναφέρονταν τυχαία (Δες αυτά τα GitHub Issues)
    • Ξαναέγραψα τον κώδικα που φορτώνει το αρχείο ρυθμίσεων για το Mac Mouse Fix ώστε να είναι πιο ανθεκτικός. Όταν συνέβαιναν σπάνια σφάλματα του συστήματος αρχείων του macOS, ο παλιός κώδικας μπορούσε μερικές φορές να νομίζει λανθασμένα ότι το αρχείο ρυθμίσεων ήταν κατεστραμμένο και να το επαναφέρει στις προεπιλογές.
  • Μειώθηκαν οι πιθανότητες ενός σφάλματος όπου το scrolling σταματά να λειτουργεί
    • Αυτό το σφάλμα δεν μπορεί να λυθεί πλήρως χωρίς βαθύτερες αλλαγές, οι οποίες πιθανόν θα προκαλούσαν άλλα προβλήματα.
      Ωστόσο, προς το παρόν, μείωσα το χρονικό παράθυρο όπου μπορεί να συμβεί ένα ‘deadlock’ στο σύστημα scrolling, κάτι που θα πρέπει τουλάχιστον να μειώσει τις πιθανότητες να αντιμετωπίσεις αυτό το σφάλμα. Αυτό κάνει επίσης το scrolling ελαφρώς πιο αποδοτικό.
    • Αυτό το σφάλμα έχει παρόμοια συμπτώματα – αλλά νομίζω διαφορετική υποκείμενη αιτία – με το σφάλμα ‘Scroll Stops Working Intermittently’ που αντιμετωπίστηκε στην προηγούμενη έκδοση 3.0.6.
