Mac Mouse Fix 3.0.7 исправляет несколько важных ошибок.

Исправления ошибок

  • Приложение снова работает на старых версиях macOS (macOS 10.15 Catalina и macOS 11 Big Sur)
    • Mac Mouse Fix 3.0.6 не удавалось включить на этих версиях macOS, потому что улучшенная функция «Назад» и «Вперёд», представленная в Mac Mouse Fix 3.0.6, пыталась использовать системные API macOS, которые были недоступны.
  • Исправлены проблемы с функцией «Назад» и «Вперёд»
    • Улучшенная функция «Назад» и «Вперёд», представленная в Mac Mouse Fix 3.0.6, теперь всегда будет использовать «основной поток» для запроса к macOS о том, какие нажатия клавиш симулировать для перехода назад и вперёд в используемом приложении.
      Это может предотвратить сбои и ненадёжное поведение в некоторых ситуациях.
  • Предпринята попытка исправить ошибку, при которой настройки случайно сбрасывались (См. эти GitHub Issues)
    • Я переписал код, который загружает файл конфигурации для Mac Mouse Fix, чтобы сделать его более надёжным. При возникновении редких ошибок файловой системы macOS старый код иногда мог ошибочно считать, что файл конфигурации повреждён, и сбрасывал его к настройкам по умолчанию.
  • Снижена вероятность ошибки, при которой прокрутка перестаёт работать
    • Эту ошибку невозможно полностью устранить без более глубоких изменений, которые, вероятно, вызовут другие проблемы.
      Однако на данный момент я сократил временное окно, в котором может произойти «взаимная блокировка» в системе прокрутки, что должно хотя бы снизить вероятность столкнуться с этой ошибкой. Это также делает прокрутку немного более эффективной.
    • Эта ошибка имеет похожие симптомы — но, я думаю, другую основную причину — с ошибкой «Прокрутка периодически перестаёт работать», которая была устранена в предыдущем релизе 3.0.6.
    • (Спасибо Joonas за диагностику!)

Спасибо всем за сообщения об ошибках!
