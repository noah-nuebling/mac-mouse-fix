ℹ️ תורגם על ידי בינה מלאכותית

הערת שחרור זו תורגמה על ידי הבינה המלאכותית Claude. היא עשויה להכיל שגיאות.
לגרסה המקורית באנגלית, לחץ כאן.

Mac Mouse Fix 3.0.7 מטפל במספר באגים חשובים.

תיקוני באגים

  • האפליקציה עובדת שוב בגרסאות macOS ישנות יותר (macOS 10.15 Catalina ו-macOS 11 Big Sur)
    • לא ניתן היה להפעיל את Mac Mouse Fix 3.0.6 בגרסאות macOS אלו מכיוון שהתכונה המשופרת ‘אחורה’ ו’קדימה’ שהוצגה ב-Mac Mouse Fix 3.0.6 ניסתה להשתמש ב-APIs של מערכת macOS שלא היו זמינים.
  • תוקנו בעיות עם תכונת ‘אחורה’ ו’קדימה’
    • התכונה המשופרת ‘אחורה’ ו’קדימה’ שהוצגה ב-Mac Mouse Fix 3.0.6 תשתמש כעת תמיד ב-‘main thread’ כדי לשאול את macOS אילו לחיצות מקשים לדמות כדי לחזור אחורה ולהתקדם קדימה באפליקציה שבה אתה משתמש.
      זה יכול למנוע קריסות והתנהגות לא אמינה במצבים מסוימים.
  • ניסיון לתקן באג שבו ההגדרות אופסו באופן אקראי (ראה GitHub Issues אלו)
    • כתבתי מחדש את הקוד שטוען את קובץ התצורה עבור Mac Mouse Fix כדי להיות יותר חזק. כאשר התרחשו שגיאות נדירות במערכת הקבצים של macOS, הקוד הישן יכול היה לפעמים לחשוב בטעות שקובץ התצורה פגום ולאפס אותו לברירת המחדל.
  • הפחתת הסיכויים לבאג שבו הגלילה מפסיקה לעבוד
    • לא ניתן לפתור באג זה במלואו ללא שינויים עמוקים יותר, שכנראה יגרמו לבעיות אחרות.
      עם זאת, לעת עתה, הפחתתי את חלון הזמן שבו יכול לקרות ‘deadlock’ במערכת הגלילה, מה שאמור לפחות להוריד את הסיכויים להיתקל בבאג זה. זה גם הופך את הגלילה ליעילה מעט יותר.
    • לבאג זה יש תסמינים דומים – אבל אני חושב שיש לו סיבה בסיסית שונה – לבאג ‘הגלילה מפסיקה לעבוד לסירוגין’ שטופל בגרסה הקודמת 3.0.6.
    • (תודה ל-Joonas על האבחון!)

תודה לכולם על דיווח הבאגים!
