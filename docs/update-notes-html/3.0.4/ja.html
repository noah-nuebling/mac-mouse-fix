Mac Mouse Fix 3.0.4 はプライバシー、効率性、信頼性を向上させます。
新しいオフラインライセンスシステムを導入し、いくつかの重要なバグを修正しました。

プライバシーと効率性の強化

3.0.4では、インターネット接続を可能な限り最小限に抑える新しいオフラインライセンス検証システムを導入しました。
これによりプライバシーが向上し、コンピュータのシステムリソースを節約できます。
ライセンス認証後は、アプリが100%オフラインで動作するようになりました!

詳細はこちらをクリック

以前のバージョンでは、起動時に毎回オンラインでライセンスを検証していたため、サードパーティのサーバー(GitHubとGumroad)に接続ログが保存される可能性がありました。新しいシステムでは不要な接続を排除し、初回のライセンス認証後は、ローカルのライセンスデータが破損した場合のみインターネットに接続します。

私自身がユーザーの行動を記録したことは一度もありませんが、以前のシステムでは理論上、サードパーティのサーバーがIPアドレスや接続時刻をログに記録できる可能性がありました。また、Gumroadはライセンスキーをログに記録し、Mac Mouse Fix購入時に記録された個人情報と関連付けることも可能でした。

元のライセンスシステムを構築した際、これらの微妙なプライバシー問題を考慮していませんでしたが、今ではMac Mouse Fixは可能な限りプライベートでインターネット不要になりました!

Gumroadのプライバシーポリシーと私のGitHubコメントもご覧ください。

バグ修正

  • 「Spaces & Mission Control」で「クリック&ドラッグ」を使用した際に、macOSがフリーズすることがあるバグを修正しました。
  • Mac Mouse Fixの「Mission Control」などの「クリック」アクションを使用した際に、システム設定のキーボードショートカットが削除されることがあるバグを修正しました。
  • アプリを既に購入済みのユーザーに対して、アプリが動作しなくなり「無料期間が終了しました」という通知が表示されるバグを修正しました。
    • このバグでご迷惑をおかけした方には、心よりお詫び申し上げます。こちらから返金を申請できます。
  • アプリケーションがメインウィンドウを取得する方法を改善し、「ライセンス認証」画面が表示されないことがあるバグを修正した可能性があります。

使いやすさの向上

  • 「ライセンス認証」画面のテキストフィールドにスペースや改行を入力できないようにしました。
    • Gumroadのメールからライセンスキーをコピーする際に、隠れた改行を誤って選択してしまうことが多く、混乱の原因となっていました。
  • このアップデートノートは、英語以外のユーザー向けに自動翻訳されています(Claude提供)。お役に立てれば幸いです!問題があればお知らせください。これは、私が過去1年間開発してきた新しい翻訳システムの最初の一歩です。

macOS 10.14 Mojaveの(非公式)サポート終了

Mac Mouse Fix 3は公式にはmacOS 11 Big Sur以降をサポートしています。ただし、いくつかの不具合やグラフィックの問題を受け入れる場合、Mac Mouse Fix 3.0.3以前はmacOS 10.14.4 Mojaveでも使用できました。

Mac Mouse Fix 3.0.4ではそのサポートを終了し、macOS 10.15 Catalina以降が必要になりました。
ご不便をおかけして申し訳ございません。この変更により、最新のSwift機能を使用して改善されたライセンスシステムを実装できるようになりました。MojaveユーザーはMac Mouse Fix 3.0.3またはMac Mouse Fix 2の最新バージョンを引き続き使用できます。皆様にとって良い解決策となることを願っています。

内部的な改善

  • 新しい「MFDataClass」システムを実装し、Mac Mouse Fixの設定ファイルを人間が読み書きできる状態を保ちながら、より強力なデータモデリングを可能にしました。
  • Gumroad以外の決済プラットフォームを追加するためのサポートを構築しました。将来的には、ローカライズされた決済や、さまざまな国での販売が可能になるかもしれません。
  • ログ機能を改善し、再現が難しいバグを経験したユーザー向けに、より効果的な「デバッグビルド」を作成できるようにしました。
  • その他多くの小さな改善とクリーンアップ作業を行いました。
