บันทึกการเปิดตัวนี้ได้รับการแปลโดย Claude AI อาจมีข้อผิดพลาดบางประการ
สำหรับเวอร์ชันภาษาอังกฤษต้นฉบับ คลิกที่นี่

Mac Mouse Fix 3.0.4 ปรับปรุงความเป็นส่วนตัว ประสิทธิภาพ และความน่าเชื่อถือ
เวอร์ชันนี้นำเสนอระบบใบอนุญาตแบบออฟไลน์ใหม่ และแก้ไขบั๊กสำคัญหลายจุด

ความเป็นส่วนตัวและประสิทธิภาพที่ดีขึ้น

3.0.4 นำเสนอระบบตรวจสอบใบอนุญาตแบบออฟไลน์ใหม่ที่ลดการเชื่อมต่ออินเทอร์เน็ตให้น้อยที่สุดเท่าที่จะเป็นไปได้
สิ่งนี้ช่วยปรับปรุงความเป็นส่วนตัวและประหยัดทรัพยากรระบบของคอมพิวเตอร์ของคุณ
เมื่อมีใบอนุญาตแล้ว แอปจะทำงานแบบออฟไลน์ 100%!

คลิกที่นี่เพื่อดูรายละเอียดเพิ่มเติม

เวอร์ชันก่อนหน้านี้ตรวจสอบใบอนุญาตออนไลน์ทุกครั้งที่เปิดแอป ซึ่งอาจทำให้บันทึกการเชื่อมต่อถูกเก็บไว้โดยเซิร์ฟเวอร์บุคคลที่สาม (GitHub และ Gumroad) ระบบใหม่ขจัดการเชื่อมต่อที่ไม่จำเป็น – หลังจากการเปิดใช้งานใบอนุญาตครั้งแรก แอปจะเชื่อมต่ออินเทอร์เน็ตก็ต่อเมื่อข้อมูลใบอนุญาตในเครื่องเสียหายเท่านั้น

แม้ว่าจะไม่เคยมีการบันทึกพฤติกรรมผู้ใช้โดยตัวผมเองเลย แต่ระบบเดิมในทางทฤษฎีอาจทำให้เซิร์ฟเวอร์บุคคลที่สามสามารถบันทึกที่อยู่ IP และเวลาการเชื่อมต่อได้ Gumroad ยังสามารถบันทึกคีย์ใบอนุญาตของคุณและอาจเชื่อมโยงกับข้อมูลส่วนตัวใดๆ ที่พวกเขาบันทึกเกี่ยวกับคุณเมื่อคุณซื้อ Mac Mouse Fix

ผมไม่ได้คำนึงถึงปัญหาความเป็นส่วนตัวที่ละเอียดอ่อนเหล่านี้เมื่อสร้างระบบใบอนุญาตเดิม แต่ตอนนี้ Mac Mouse Fix มีความเป็นส่วนตัวและไม่ต้องใช้อินเทอร์เน็ตมากที่สุดเท่าที่จะเป็นไปได้!

ดูเพิ่มเติมที่ นโยบายความเป็นส่วนตัวของ Gumroad และความคิดเห็นบน GitHub ของผม

การแก้ไขบั๊ก

  • แก้ไขบั๊กที่ macOS บางครั้งค้างเมื่อใช้ ‘Click and Drag’ สำหรับ ‘Spaces & Mission Control’
  • แก้ไขบั๊กที่แป้นพิมพ์ลัดใน System Settings บางครั้งถูกลบเมื่อใช้การกระทำ ‘Click’ ของ Mac Mouse Fix เช่น ‘Mission Control’
  • แก้ไขบั๊กที่แอปบางครั้งหยุดทำงานและแสดงการแจ้งเตือนว่า ‘Free days are over’ ให้กับผู้ใช้ที่ซื้อแอปไปแล้ว
    • หากคุณประสบปัญหานี้ ผมขออภัยอย่างจริงใจสำหรับความไม่สะดวก คุณสามารถขอคืนเงินได้ที่นี่
  • ปรับปรุงวิธีที่แอปพลิเคชันดึงหน้าต่างหลัก ซึ่งอาจแก้ไขบั๊กที่หน้าจอ ‘Activate License’ บางครั้งไม่ปรากฏขึ้น

การปรับปรุงความสะดวกในการใช้งาน

  • ทำให้ไม่สามารถใส่ช่องว่างและการขึ้นบรรทัดใหม่ในช่องข้อความบนหน้าจอ ‘Activate License’ ได้
    • นี่เป็นจุดที่สร้างความสับสนบ่อยครั้ง เพราะง่ายมากที่จะเลือกการขึ้นบรรทัดใหม่ที่ซ่อนอยู่โดยไม่ตั้งใจเมื่อคัดลอกคีย์ใบอนุญาตจากอีเมลของ Gumroad
  • บันทึกการอัปเดตนี้ถูกแปลโดยอัตโนมัติสำหรับผู้ใช้ที่ไม่ใช่ภาษาอังกฤษ (ขับเคลื่อนโดย Claude) หวังว่าจะเป็นประโยชน์! หากคุณพบปัญหาใดๆ โปรดแจ้งให้ผมทราบ นี่เป็นภาพรวมแรกของระบบแปลใหม่ที่ผมพัฒนามาตลอดปีที่ผ่านมา

ยกเลิกการรองรับ (ไม่เป็นทางการ) สำหรับ macOS 10.14 Mojave

Mac Mouse Fix 3 รองรับอย่างเป็นทางการสำหรับ macOS 11 Big Sur และใหม่กว่า อย่างไรก็ตาม สำหรับผู้ใช้ที่ยินดีรับปัญหาบั๊กและปัญหาด้านกราฟิกบางอย่าง Mac Mouse Fix 3.0.3 และเวอร์ชันก่อนหน้ายังสามารถใช้งานบน macOS 10.14.4 Mojave ได้

Mac Mouse Fix 3.0.4 ยกเลิกการรองรับนั้นและตอนนี้ต้องการ macOS 10.15 Catalina
ผมขออภัยสำหรับความไม่สะดวกที่เกิดจากการเปลี่ยนแปลงนี้ การเปลี่ยนแปลงนี้ทำให้ผมสามารถใช้งานระบบใบอนุญาตที่ปรับปรุงแล้วโดยใช้ฟีเจอร์ Swift สมัยใหม่ได้ ผู้ใช้ Mojave สามารถใช้ Mac Mouse Fix 3.0.3 หรือเวอร์ชันล่าสุดของ Mac Mouse Fix 2 ต่อไปได้ หวังว่านี่จะเป็นทางออกที่ดีสำหรับทุกคน

การปรับปรุงภายใต้ฝากระโปรง

  • ใช้งานระบบ ‘MFDataClass’ ใหม่ที่ช่วยให้สามารถสร้างโมเดลข้อมูลที่ทรงพลังมากขึ้น ในขณะที่ยังคงไฟล์คอนฟิกของ Mac Mouse Fix อ่านและแก้ไขได้ง่ายสำหรับมนุษย์
  • สร้างการรองรับสำหรับการเพิ่มแพลตฟอร์มการชำระเงินอื่นนอกเหนือจาก Gumroad ดังนั้นในอนาคตอาจมีการชำระเงินแบบท้องถิ่น และแอปอาจขายให้กับประเทศต่างๆ ได้
  • ปรับปรุงการบันทึกล็อกซึ่งช่วยให้ผมสามารถสร้าง “Debug Builds” ที่มีประสิทธิภาพมากขึ้นสำหรับผู้ใช้ที่ประสบบั๊กที่ยากต่อการทำซ้ำ
  • การปรับปรุงเล็กๆ น้อยๆ และงานทำความสะอาดอื่นๆ อีกมากมาย

แก้ไขด้วยความช่วยเหลืออันยอดเยี่ยมจาก Claude
