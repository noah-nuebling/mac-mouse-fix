Mac Mouse Fix 3.0.8 はUIの問題などを解決しました。

UIの問題

  • macOS 26 Tahoeで新しいデザインを無効にしました。アプリはmacOS 15 Sequoiaと同じ見た目と動作になります。
    • Appleが再設計したUI要素の一部にまだ問題があるため、この変更を行いました。例えば、「ボタン」タブの「-」ボタンが常にクリックできるわけではありませんでした。
    • macOS 26 Tahoeでは、UIが少し古く見えるかもしれません。しかし、以前と同じように完全に機能し、洗練されているはずです。
  • 「無料期間が終了しました」通知が画面右上の隅に固定されてしまうバグを修正しました。
    • 報告してくれたSashpuriさんや他の方々に感謝します!

UIの改善

  • Mac Mouse Fixのメインウィンドウの緑色の信号ボタンを無効にしました。
    • ウィンドウは手動でサイズ変更できないため、このボタンは何もしませんでした。
  • macOS 26 Tahoeで「ボタン」タブのテーブルの水平線の一部が暗すぎる問題を修正しました。
  • macOS 26 Tahoeで「ボタン」タブの「プライマリマウスボタンは使用できません」メッセージが切れてしまうことがあるバグを修正しました。
  • ドイツ語インターフェースのタイプミスを修正しました。GitHubユーザーi-am-the-slimeさんのご協力に感謝します!
  • macOS 26 TahoeでMMFウィンドウを開く際に、ウィンドウが間違ったサイズで一瞬点滅することがある問題を解決しました。

その他の変更

  • コンピュータ上で複数のMac Mouse Fixインスタンスが実行されている状態でMac Mouse Fixを有効にしようとする際の動作を改善しました。
    • Mac Mouse Fixは、他のMac Mouse Fixインスタンスをより積極的に無効化しようとするようになりました。
    • これにより、Mac Mouse Fixを有効にできなかったエッジケースが改善される可能性があります。
  • 内部的な変更とクリーンアップ。

前バージョン3.0.7の新機能もチェックしてください。
