Mac Mouse Fix 3.0.8 исправляет проблемы с интерфейсом и многое другое.

Проблемы с интерфейсом

  • Отключён новый дизайн в macOS 26 Tahoe. Теперь приложение будет выглядеть и работать так же, как в macOS 15 Sequoia.
    • Я сделал это, потому что некоторые переработанные элементы интерфейса Apple всё ещё имеют проблемы. Например, кнопки «-» на вкладке «Кнопки» не всегда были кликабельны.
    • Интерфейс может выглядеть немного устаревшим в macOS 26 Tahoe сейчас. Но он должен быть полностью функциональным и отполированным, как и раньше.
  • Исправлена ошибка, из-за которой уведомление «Бесплатные дни закончились» застревало в правом верхнем углу экрана.
    • Спасибо Sashpuri и другим за сообщение об этом!

Улучшения интерфейса

  • Отключена зелёная кнопка светофора в главном окне Mac Mouse Fix.
    • Кнопка ничего не делала, так как окно нельзя изменить вручную.
  • Исправлена проблема, из-за которой некоторые горизонтальные линии в таблице на вкладке «Кнопки» были слишком тёмными в macOS 26 Tahoe.
  • Исправлена ошибка, из-за которой сообщение «Основная кнопка мыши не может быть использована» на вкладке «Кнопки» иногда обрезалось в macOS 26 Tahoe.
  • Исправлена опечатка в немецком интерфейсе. Благодаря пользователю GitHub i-am-the-slime. Спасибо!
  • Решена проблема, из-за которой окно MMF иногда кратковременно мигало с неправильным размером при открытии окна в macOS 26 Tahoe.

Другие изменения

  • Улучшено поведение при попытке включить Mac Mouse Fix, когда на компьютере запущено несколько экземпляров Mac Mouse Fix.
    • Mac Mouse Fix теперь будет более старательно пытаться отключить другой экземпляр Mac Mouse Fix.
    • Это может улучшить крайние случаи, когда Mac Mouse Fix не удавалось включить.
  • Внутренние изменения и очистка кода.
