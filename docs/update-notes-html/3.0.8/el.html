ℹ️ Μετάφραση από AI

Αυτές οι σημειώσεις έκδοσης έχουν μεταφραστεί από το Claude AI. Ενδέχεται να περιέχουν λάθη.
Για την πρωτότυπη αγγλική έκδοση, πατήστε εδώ.

Το Mac Mouse Fix 3.0.8 επιλύει προβλήματα διεπαφής και άλλα.

Προβλήματα Διεπαφής

  • Απενεργοποιήθηκε ο νέος σχεδιασμός στο macOS 26 Tahoe. Τώρα η εφαρμογή θα έχει την εμφάνιση και τη λειτουργικότητα που είχε στο macOS 15 Sequoia.
    • Το έκανα αυτό επειδή μερικά από τα ανασχεδιασμένα στοιχεία διεπαφής της Apple εξακολουθούν να έχουν προβλήματα. Για παράδειγμα, τα κουμπιά ‘-’ στην καρτέλα ‘Κουμπιά’ δεν ήταν πάντα κλικάρισμα.
    • Η διεπαφή μπορεί να φαίνεται λίγο ξεπερασμένη στο macOS 26 Tahoe τώρα. Αλλά θα είναι πλήρως λειτουργική και γυαλισμένη όπως πριν.
  • Διορθώθηκε ένα σφάλμα όπου η ειδοποίηση ‘Οι δωρεάν ημέρες τελείωσαν’ κολλούσε στην πάνω δεξιά γωνία της οθόνης.
    • Ευχαριστώ τον Sashpuri και άλλους που το ανέφεραν!

Βελτιώσεις Διεπαφής

  • Απενεργοποιήθηκε το πράσινο κουμπί φωτεινού σηματοδότη στο κύριο παράθυρο του Mac Mouse Fix.
    • Το κουμπί δεν έκανε τίποτα, αφού το παράθυρο δεν μπορεί να αλλάξει μέγεθος χειροκίνητα.
  • Διορθώθηκε ένα πρόβλημα όπου μερικές από τις οριζόντιες γραμμές στον πίνακα της καρτέλας ‘Κουμπιά’ ήταν πολύ σκούρες στο macOS 26 Tahoe.
  • Διορθώθηκε ένα σφάλμα όπου το μήνυμα “Το κύριο κουμπί του ποντικιού δεν μπορεί να χρησιμοποιηθεί” στην καρτέλα ‘Κουμπιά’ μερικές φορές κοβόταν στο macOS 26 Tahoe.
  • Διορθώθηκε ένα τυπογραφικό λάθος στη γερμανική διεπαφή. Ευγενική προσφορά του χρήστη GitHub i-am-the-slime. Ευχαριστώ!
  • Επιλύθηκε ένα πρόβλημα όπου το παράθυρο του MMF μερικές φορές αναβόσβηνε σύντομα στο λάθος μέγεθος κατά το άνοιγμα του παραθύρου στο macOS 26 Tahoe.

Άλλες Αλλαγές

  • Βελτιωμένη συμπεριφορά όταν προσπαθείς να ενεργοποιήσεις το Mac Mouse Fix ενώ τρέχουν πολλαπλές εκδόσεις του Mac Mouse Fix στον υπολογιστή.
    • Το Mac Mouse Fix θα προσπαθεί τώρα να απενεργοποιήσει την άλλη έκδοση του Mac Mouse Fix πιο επιμελώς.
    • Αυτό μπορεί να βελτιώσει ακραίες περιπτώσεις όπου το Mac Mouse Fix δεν μπορούσε να ενεργοποιηθεί.
  • Αλλαγές και καθαρισμός στο παρασκήνιο.

Δες επίσης τι νέο υπάρχει στην προηγούμενη έκδοση 3.0.7.
