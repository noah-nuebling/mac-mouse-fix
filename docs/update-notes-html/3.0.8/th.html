บันทึกการเปิดตัวนี้ได้รับการแปลโดย Claude AI อาจมีข้อผิดพลาดบางประการ
สำหรับเวอร์ชันภาษาอังกฤษต้นฉบับ คลิกที่นี่

Mac Mouse Fix 3.0.8 แก้ไขปัญหา UI และอื่นๆ

ปัญหา UI

  • ปิดการใช้งานดีไซน์ใหม่บน macOS 26 Tahoe แอปจะมีลักษณะและทำงานเหมือนกับบน macOS 15 Sequoia
    • ทำเช่นนี้เพราะองค์ประกอบ UI ที่ Apple ออกแบบใหม่บางส่วนยังมีปัญหา ตัวอย่างเช่น ปุ่ม ‘-’ ในแท็บ ‘Buttons’ ไม่สามารถคลิกได้เสมอ
    • UI อาจดูล้าสมัยเล็กน้อยบน macOS 26 Tahoe ตอนนี้ แต่ควรทำงานได้อย่างสมบูรณ์และสวยงามเหมือนเดิม
  • แก้ไขบั๊กที่การแจ้งเตือน ‘Free days are over’ จะติดอยู่ที่มุมบนขวาของหน้าจอ
    • ขอบคุณ Sashpuri และคนอื่นๆ ที่รายงาน!

ปรับปรุง UI

  • ปิดการใช้งานปุ่ม traffic light สีเขียวในหน้าต่างหลักของ Mac Mouse Fix
    • ปุ่มนี้ไม่ได้ทำอะไร เนื่องจากหน้าต่างไม่สามารถปรับขนาดด้วยตนเองได้
  • แก้ไขปัญหาที่เส้นแนวนอนบางเส้นในตารางบนแท็บ ‘Buttons’ มีสีเข้มเกินไปภายใต้ macOS 26 Tahoe
  • แก้ไขบั๊กที่ข้อความ “Primary Mouse Button can’t be used” บนแท็บ ‘Buttons’ บางครั้งถูกตัดออกภายใต้ macOS 26 Tahoe
  • แก้ไขข้อผิดพลาดในอินเทอร์เฟซภาษาเยอรมัน ขอบคุณผู้ใช้ GitHub i-am-the-slime ขอบคุณ!
  • แก้ไขปัญหาที่หน้าต่าง MMF บางครั้งกะพริบสั้นๆ ในขนาดที่ไม่ถูกต้องเมื่อเปิดหน้าต่างบน macOS 26 Tahoe

การเปลี่ยนแปลงอื่นๆ

  • ปรับปรุงพฤติกรรมเมื่อพยายามเปิดใช้งาน Mac Mouse Fix ในขณะที่มี Mac Mouse Fix หลายอินสแตนซ์ทำงานอยู่บนคอมพิวเตอร์
    • Mac Mouse Fix จะพยายามปิดการใช้งานอินสแตนซ์อื่นของ Mac Mouse Fix อย่างขยันขันแข็งมากขึ้น
    • สิ่งนี้อาจปรับปรุงกรณีพิเศษที่ Mac Mouse Fix ไม่สามารถเปิดใช้งานได้
  • การเปลี่ยนแปลงและการจัดระเบียบภายใน

ดูสิ่งใหม่ในเวอร์ชันก่อนหน้า 3.0.7 ได้ด้วย
