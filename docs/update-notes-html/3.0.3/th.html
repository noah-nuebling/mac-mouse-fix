บันทึกการเปิดตัวนี้ได้รับการแปลโดย Claude AI อาจมีข้อผิดพลาดบางประการ
สำหรับเวอร์ชันภาษาอังกฤษต้นฉบับ คลิกที่นี่

Mac Mouse Fix 3.0.3 พร้อมใช้งานสำหรับ macOS 15 Sequoia แล้ว รุ่นนี้แก้ไขปัญหาเสถียรภาพและปรับปรุงในรายละเอียดต่างๆ

รองรับ macOS 15 Sequoia

แอปทำงานได้อย่างถูกต้องบน macOS 15 Sequoia แล้ว!

  • แอนิเมชัน UI ส่วนใหญ่ใช้งานไม่ได้บน macOS 15 Sequoia ตอนนี้ทุกอย่างทำงานได้ปกติแล้ว!
  • ซอร์สโค้ดสามารถ build ได้บน macOS 15 Sequoia แล้ว ก่อนหน้านี้มีปัญหากับ Swift compiler ที่ทำให้แอป build ไม่ได้

แก้ไขปัญหาการ crash ขณะเลื่อน

ตั้งแต่ Mac Mouse Fix 3.0.2 มีรายงานหลายครั้งว่า Mac Mouse Fix ปิดและเปิดตัวเองซ้ำๆ ขณะเลื่อน ซึ่งเกิดจากการ crash ของแอปพื้นหลัง ‘Mac Mouse Fix Helper’ อัปเดตนี้พยายามแก้ไข crash เหล่านี้ด้วยการเปลี่ยนแปลงดังนี้:

  • กลไกการเลื่อนจะพยายามกู้คืนและทำงานต่อแทนที่จะ crash เมื่อพบกรณีพิเศษที่ดูเหมือนจะนำไปสู่ crash เหล่านี้

  • ฉันเปลี่ยนวิธีจัดการกับสถานะที่ไม่คาดคิดในแอปโดยทั่วไป: แทนที่จะ crash ทันที แอปจะพยายามกู้คืนจากสถานะที่ไม่คาดคิดในหลายกรณี

    • การเปลี่ยนแปลงนี้ช่วยแก้ไขปัญหา scroll crash ที่กล่าวถึงข้างต้น และอาจป้องกัน crash อื่นๆ ได้ด้วย

หมายเหตุ: ฉันไม่สามารถทำให้เกิด crash เหล่านี้บนเครื่องของฉันได้เลย และยังไม่แน่ใจว่าอะไรเป็นสาเหตุ แต่จากรายงานที่ได้รับ อัปเดตนี้น่าจะป้องกัน crash ได้ทั้งหมด หากคุณยังพบ crash ขณะเลื่อน หรือหากคุณเคยพบ crash ใน 3.0.2 จะเป็นประโยชน์มากหากคุณแชร์ประสบการณ์และข้อมูลวินิจฉัยใน GitHub Issue #988 ซึ่งจะช่วยให้ฉันเข้าใจปัญหาและปรับปรุง Mac Mouse Fix ได้ ขอบคุณ!

แก้ไขปัญหาการเลื่อนสะดุด

ใน 3.0.2 ฉันเปลี่ยนวิธีที่ Mac Mouse Fix ส่ง scroll event ไปยังระบบเพื่อพยายามลดการเลื่อนสะดุดที่น่าจะเกิดจากปัญหากับ VSync API ของ Apple

อย่างไรก็ตาม หลังจากทดสอบและรับฟีดแบ็กอย่างละเอียดมากขึ้น ดูเหมือนว่ากลไกใหม่ใน 3.0.2 ทำให้การเลื่อนลื่นขึ้นในบางสถานการณ์ แต่สะดุดมากขึ้นในสถานการณ์อื่นๆ โดยเฉพาะใน Firefox ดูเหมือนจะแย่ลงอย่างเห็นได้ชัด
โดยรวมแล้ว ไม่ชัดเจนว่ากลไกใหม่ช่วยลดการเลื่อนสะดุดได้จริงในทุกกรณี และอาจมีส่วนทำให้เกิด scroll crash ที่กล่าวถึงข้างต้นด้วย

ด้วยเหตุนี้ ฉันจึงปิดกลไกใหม่และเปลี่ยนกลไก VSync สำหรับ scroll event กลับไปเป็นแบบเดียวกับใน Mac Mouse Fix 3.0.0 และ 3.0.1

ดูข้อมูลเพิ่มเติมได้ที่ GitHub Issue #875

การคืนเงิน

ฉันขอโทษสำหรับปัญหาที่เกี่ยวข้องกับการเปลี่ยนแปลงการเลื่อนใน 3.0.1 และ 3.0.2 ฉันประเมินปัญหาที่จะตามมาต่ำเกินไป และช้าในการแก้ไขปัญหาเหล่านี้ ฉันจะพยายามอย่างเต็มที่เพื่อเรียนรู้จากประสบการณ์นี้และระมัดระวังมากขึ้นกับการเปลี่ยนแปลงเช่นนี้ในอนาคต ฉันยังอยากเสนอการคืนเงินให้กับทุกคนที่ได้รับผลกระทบ เพียงคลิกที่นี่หากสนใจ

กลไกอัปเดตที่ฉลาดขึ้น

การเปลี่ยนแปลงเหล่านี้นำมาจาก Mac Mouse Fix 2.2.4 และ 2.2.5 ดูบันทึกการเผยแพร่เพื่อเรียนรู้รายละเอียดเพิ่มเติม นี่คือสรุป:

  • มีกลไกใหม่ที่ฉลาดขึ้นในการตัดสินใจว่าจะแสดงอัปเดตใดให้ผู้ใช้
  • เปลี่ยนจากการใช้ Sparkle 1.26.0 update framework เป็น Sparkle 1.27.3 เวอร์ชันล่าสุด
  • หน้าต่างที่แอปแสดงเพื่อแจ้งให้คุณทราบว่ามี Mac Mouse Fix เวอร์ชันใหม่รองรับ JavaScript แล้ว ซึ่งช่วยให้จัดรูปแบบบันทึกอัปเดตได้สวยงามขึ้น

การปรับปรุงและแก้ไขบั๊กอื่นๆ

  • แก้ไขปัญหาที่ราคาแอปและข้อมูลที่เกี่ยวข้องแสดงไม่ถูกต้องในแท็บ ‘About’ ในบางกรณี
  • แก้ไขปัญหาที่กลไกการซิงค์การเลื่อนแบบลื่นกับอัตราการรีเฟรชของจอแสดงผลทำงานไม่ถูกต้องขณะใช้หลายจอ
  • การปรับปรุงและทำความสะอาดเบื้องหลังเล็กน้อยมากมาย
