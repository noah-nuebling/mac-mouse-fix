ℹ️ Μετάφραση από AI

Αυτές οι σημειώσεις έκδοσης έχουν μεταφραστεί από το Claude AI. Ενδέχεται να περιέχουν λάθη.
Για την πρωτότυπη αγγλική έκδοση, πατήστε εδώ.

Το Mac Mouse Fix 3.1.0 Beta 1 φέρνει ένα εξαιρετικό νέο σύστημα μετάφρασης, το οποίο βρισκόταν υπό ανάπτυξη για αρκετό καιρό.
Υπάρχουν επίσης κάποιες βελτιώσεις στη διεπαφή χρήστη.

Νέο σύστημα μετάφρασης!

  • Είναι πλέον πολύ εύκολο να συνεισφέρεις υψηλής ποιότητας μεταφράσεις στο Mac Mouse Fix, χωρίς να απαιτείται καμία τεχνική γνώση.
  • Υπάρχουν πολλά σχολιασμένα στιγμιότυπα οθόνης και σχόλια, καθώς και μια εξαιρετική λειτουργία αναζήτησης, ώστε οι μεταφραστές να μπορούν εύκολα να καταλάβουν ακριβώς πώς εμφανίζονται οι μεταφράσεις τους στην εφαρμογή.
    • Πιστεύω ότι τα στιγμιότυπα οθόνης και η διεπαφή μετάφρασης (εφαρμογή Xcloc Editor) είναι της υψηλότερης ποιότητας στην αγορά, και ελπίζω ότι θα σε βοηθήσουν να παρέχεις εξαιρετικές μεταφράσεις!
  • Σχεδόν όλα τα μέρη του έργου είναι πλέον μεταφράσιμα, συμπεριλαμβανομένου του ‘Οδηγού Καταγεγραμμένων Κουμπιών’, του GitHub Readme, της Ιστοσελίδας και άλλων.
    • Όλα αυτά τα μέρη του έργου μπορούν να μεταφραστούν από ένα κεντρικό σημείο με εύκολη διασταύρωση αναφορών χρησιμοποιώντας την εφαρμογή Xcloc Editor, ώστε να είναι εύκολο για τους μεταφραστές να διατηρούν τα πάντα συνεπή.
  • Το σύστημα είναι σχεδιασμένο να είναι πολύ εύκολο στη συντήρηση, οπότε καθώς το Mac Mouse Fix εξελίσσεται και προστίθενται νέες οθόνες ή κείμενα, όλα θα ενημερώνονται αυτόματα.
  • Αποποίηση ευθύνης: Χρησιμοποιώ πολύ τη λέξη ‘εύκολο’. Οι ροές εργασίας είναι εύκολες, αλλά η σκέψη για εξαιρετικές μεταφράσεις είναι φυσικά ακόμα πολλή σκληρή και αξιοσέβαστη δουλειά! Απλά υπάρχουν λιγότερα εμπόδια στο δρόμο τώρα.

Ωστόσο, όλες οι γλώσσες χρειάζονται ακόμα δουλειά, καθώς πολλά νέα πράγματα έχουν γίνει μεταφράσιμα.

Επομένως:

[!TIP] Ρίξε μια ματιά στον νέο Οδηγό Μετάφρασης αν θέλεις να βοηθήσεις να φέρεις εξαιρετικές μεταφράσεις στους χρήστες του Mac Mouse Fix σε όλο τον κόσμο!

Άλλες αλλαγές

Όταν χρησιμοποιείς το Mac Mouse Fix στα Αγγλικά, δεν έχει αλλάξει πολλά, αλλά μερικά πράγματα έχουν ενημερωθεί στη διεπαφή χρήστη, και πολλά έχουν αλλάξει στο παρασκήνιο, για να βοηθήσουν να γίνουν οι μεταφράσεις καλύτερες:

  • Βελτιωμένη εμφάνιση για τις μικρές αναδυόμενες ειδοποιήσεις.
  • Πιο σαφείς συμβουλές εργαλείων και σφάλματα σε διάφορα σημεία.
  • Το πλήκτρο escape μπορεί πλέον να χρησιμοποιηθεί παντού για να απορρίψεις ειδοποιήσεις, φύλλα και αναδυόμενα παράθυρα.
  • Ένας σύνδεσμος ‘Βοήθησε στη Μετάφραση’ έχει προστεθεί στην καρτέλα Σχετικά.
  • Το πλάτος των προβολών και των αναδυόμενων παραθύρων έχει προσαρμοστεί για να φαίνεται εξαιρετικό σε διάφορες γλώσσες.
  • Καλύτερη διάταξη κειμένου για τα Κινέζικα και τα Κορεάτικα, που είχαν προβλήματα αναδίπλωσης κειμένου σε μερικές από τις μικρές αναδυόμενες ειδοποιήσεις.
  • Αντικατάσταση της βιβλιοθήκης ανάλυσης Markdown για να διορθωθούν κάποια σφάλματα μορφοποίησης στα Κινέζικα και τα Κορεάτικα.
  • Πολλές ακόμα αλλαγές και βελτιώσεις στο παρασκήνιο.
