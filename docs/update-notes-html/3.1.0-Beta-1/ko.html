Mac Mouse Fix 3.0.1 베타 1은 오랫동안 작업해온 훌륭한 새 번역 시스템을 제공합니다.
또한 몇 가지 UI 개선 사항도 포함되어 있습니다.

새로운 현지화 시스템!

  • 이제 기술적 배경 지식이 전혀 없어도 Mac Mouse Fix에 고품질 번역을 기여하는 것이 매우 쉬워졌습니다.
  • 주석이 달린 스크린샷과 설명, 그리고 훌륭한 검색 기능이 많이 제공되어 번역자들이 자신의 번역이 앱에서 어떻게 표시되는지 정확히 이해할 수 있습니다.
    • 스크린샷과 번역 UI(Xcloc Editor 앱)가 시장에서 최고 품질이라고 생각하며, 여러분이 훌륭한 번역을 제공하는 데 도움이 되기를 바랍니다!
  • 이제 ‘캡처된 버튼 가이드’, GitHub Readme, 웹사이트 등을 포함하여 프로젝트의 거의 모든 부분을 번역할 수 있습니다.
    • 프로젝트의 이러한 모든 부분을 Xcloc Editor 앱을 사용하여 쉬운 상호 참조와 함께 하나의 중앙 집중식 장소에서 번역할 수 있어, 번역자가 모든 것을 일관되게 유지하기 쉽습니다.
  • 시스템은 유지 관리가 매우 쉽도록 설정되어 있어, Mac Mouse Fix가 발전하고 새로운 화면이나 텍스트가 추가되면 모든 것이 자동으로 업데이트됩니다.
  • 면책 조항: ’쉽다’는 단어를 많이 사용합니다. 작업 흐름은 쉽지만, 훌륭한 번역을 생각해내는 것은 물론 여전히 많은 노력과 존중받을 만한 작업입니다! 이제 방해 요소가 줄어들었을 뿐입니다.

기타 변경 사항

Mac Mouse Fix를 영어로 사용할 때는 크게 변경된 것이 없지만, UI에서 몇 가지가 업데이트되었고 번역을 더 잘 만들기 위해 내부적으로 많은 것이 변경되었습니다:

  • 작은 팝업 알림의 외관이 개선되었습니다.
  • 여러 곳에서 툴팁과 오류 메시지가 더 명확해졌습니다.
  • 이제 모든 곳에서 ESC 키를 사용하여 알림, 시트, 팝업을 닫을 수 있습니다.
  • 정보 탭에 ‘번역 돕기’ 링크가 추가되었습니다.
  • 다양한 언어에서 멋지게 보이도록 뷰와 팝업의 너비가 조정되었습니다.
  • 일부 작은 팝업 알림에서 텍스트 줄 바꿈 문제가 있던 중국어와 한국어의 텍스트 레이아웃이 개선되었습니다.
  • 중국어와 한국어의 일부 서식 버그를 수정하기 위해 Markdown 파싱 라이브러리를 교체했습니다.
  • 그 외 많은 내부 변경 사항 및 개선 사항이 있습니다.
