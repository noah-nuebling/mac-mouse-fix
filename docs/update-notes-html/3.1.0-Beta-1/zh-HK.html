Mac Mouse Fix 3.0.1 Beta 1 帶來了一個全新的翻譯系統,這個系統已經開發了一段時間。
同時還有一些使用者介面的改進。

全新本地化系統!

  • 現在為 Mac Mouse Fix 貢獻高質量翻譯變得超級簡單,完全不需要任何技術背景。
  • 提供了大量附註的螢幕截圖和註釋,以及強大的搜尋功能,讓譯者可以輕鬆準確地了解他們的翻譯在應用程式中的顯示效果。
    • 我認為這些螢幕截圖和翻譯介面(Xcloc Editor 應用程式)是市場上質量最高的,希望它們能幫助你提供出色的翻譯!
  • 現在幾乎所有專案部分都可以翻譯,包括「已捕捉按鈕指南」、GitHub 說明文件、網站等等。
    • 所有這些專案部分都可以在一個集中的地方進行翻譯,並使用 Xcloc Editor 應用程式輕鬆交叉參照,讓譯者能夠輕鬆保持一切的一致性。
  • 這個系統設計得非常易於維護,所以隨著 Mac Mouse Fix 的發展和新畫面或文字的加入,一切都會自動更新。
  • 免責聲明:我經常使用「簡單」這個詞。工作流程是簡單的,但想出出色的翻譯當然仍然是大量艱苦且值得尊敬的工作!只是現在障礙少了很多。

其他變更

在英語環境下使用 Mac Mouse Fix 時,變化不大,但使用者介面有一些更新,而且底層進行了大量調整,以幫助改善翻譯:

  • 改進了小型彈出通知的外觀。
  • 在各個地方提供了更清晰的工具提示和錯誤訊息。
  • 現在可以在任何地方使用 Escape 鍵來關閉通知、工作表和彈出視窗。
  • 在「關於」標籤頁中加入了「協助翻譯」連結。
  • 調整了視圖和彈出視窗的寬度,以在各種語言中呈現出色的效果。
  • 改善了中文和韓文的文字排版,這些語言在某些小型彈出通知中存在文字換行問題。
  • 更換了 Markdown 解析庫,以修復中文和韓文中的一些格式錯誤。
  • 更多底層變更和改進。
