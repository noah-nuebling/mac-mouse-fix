Mac Mouse Fix 3.0.0 Beta 1 は、しばらく開発を進めてきた素晴らしい新しい翻訳システムを導入します。
また、いくつかのUI改善も含まれています。

新しいローカライゼーションシステム!

  • 技術的な知識が一切不要で、Mac Mouse Fixに高品質な翻訳を提供することが非常に簡単になりました。
  • たくさんの注釈付きスクリーンショットやコメント、優れた検索機能があるため、翻訳者は自分の翻訳がアプリ内でどのように表示されるかを正確に理解できます。
    • スクリーンショットと翻訳UI(Xcloc Editorアプリ)は市場で最高品質だと思っており、素晴らしい翻訳の提供に役立つことを願っています!
  • 「キャプチャされたボタンガイド」、GitHub Readme、ウェブサイトなど、プロジェクトのほぼすべての部分が翻訳可能になりました。
    • これらすべてのプロジェクト部分は、Xcloc Editorアプリを使用して簡単に相互参照できる一元化された場所から翻訳できるため、翻訳者がすべてを一貫性を保つことが容易です。
  • このシステムは非常にメンテナンスしやすく設計されているため、Mac Mouse Fixが進化し、新しい画面やテキストが追加されても、すべてが自動的に更新されます。
  • 免責事項:「簡単」という言葉をよく使っていますが、ワークフローは簡単ですが、素晴らしい翻訳を考えることはもちろん依然として多くの大変で尊敬すべき作業です!ただ、今は障壁が少なくなっただけです。

その他の変更

Mac Mouse Fixを英語で使用する場合、大きな変更はありませんが、UIのいくつかが更新され、翻訳をより良くするために内部で多くの変更が行われました:

  • 小さなポップアップ通知の外観を洗練しました。
  • 様々な場所でツールチップとエラーをより明確にしました。
  • エスケープキーで通知、シート、ポップアップを閉じることができるようになりました。
  • 「翻訳を手伝う」リンクを「について」タブに追加しました。
  • 様々な言語で美しく見えるように、ビューとポップアップの幅を調整しました。
  • 一部の小さなポップアップ通知でテキストの折り返しに問題があった中国語と韓国語のテキストレイアウトを改善しました。
  • 中国語と韓国語の一部のフォーマットバグを修正するため、Markdown解析ライブラリを置き換えました。
  • その他多くの内部変更と改善。
