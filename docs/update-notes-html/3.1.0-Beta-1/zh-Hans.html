Mac Mouse Fix 3.1.0 Beta 1 带来了一个全新的翻译系统,这个系统已经开发了一段时间。
同时还有一些界面改进。

全新的本地化系统!

  • 现在为 Mac Mouse Fix 贡献高质量翻译变得超级简单,无需任何技术背景。
  • 提供了大量带注释的截图和说明,以及强大的搜索功能,译者可以轻松准确地了解他们的翻译在应用中的显示效果。
    • 我认为这些截图和翻译界面(Xcloc Editor 应用)是市场上质量最高的,希望它们能帮助你提供出色的翻译!
  • 现在项目的几乎所有部分都可以翻译了,包括”捕获按钮指南”、GitHub Readme、网站等等。
    • 项目的所有这些部分都可以在一个集中的地方使用 Xcloc Editor 应用进行翻译,并且可以轻松交叉引用,因此译者可以轻松保持所有内容的一致性。
  • 该系统的设计非常易于维护,因此随着 Mac Mouse Fix 的发展和新屏幕或文本的添加,一切都会自动更新。
  • 免责声明:我经常使用”简单”这个词。工作流程是简单的,但想出优秀的翻译当然仍然需要大量艰苦且值得尊敬的工作!只是现在障碍更少了。

其他变化

在英语环境下使用 Mac Mouse Fix 时,变化不大,但界面上有一些更新,底层也进行了大量调整,以帮助改进翻译:

  • 改进了小弹出通知的外观。
  • 在多个地方提供了更清晰的工具提示和错误信息。
  • 现在可以在任何地方使用 Esc 键来关闭通知、表单和弹出窗口。
  • 在”关于”标签页中添加了”帮助翻译”链接。
  • 调整了视图和弹出窗口的宽度,以便在各种语言中都能呈现出色的效果。
  • 改进了中文和韩文的文本布局,这两种语言在某些小弹出通知中存在文本换行问题。
  • 更换了 Markdown 解析库,以修复中文和韩文中的一些格式错误。
  • 更多底层变化和改进。
