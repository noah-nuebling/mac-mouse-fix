Επιστροφή στο Mac Mouse Fix 2, αφού δοκιμάσεις το Mac Mouse Fix 3

Μπορείς να δοκιμάσεις το MMF 3 και να επιστρέψεις στο MMF 2 ανά πάσα στιγμή.
Ωστόσο, η εγκατάσταση του MMF 3 θα διαγράψει τις ρυθμίσεις του MMF 2.
Δες πώς να αποφύγεις την απώλεια των ρυθμίσεων του MMF 2 όταν δοκιμάζεις το MMF 3:

Πριν την ενημέρωση στο MMF 3:

  1. Άνοιξε την οθόνη ‘Περισσότερα…’ στο MMF 2, μετά κάνε κλικ στο λογότυπο του MMF στην πάνω αριστερή γωνία ενώ κρατάς πατημένα τα πλήκτρα Command (⌘) και Shift (⇧) στο πληκτρολόγιό σου. Αυτό θα εμφανίσει ένα αρχείο με το όνομα ‘config.plist’ στο Finder. Αυτό το αρχείο περιέχει όλες τις ρυθμίσεις σου.
  2. Αποθήκευσε ένα αντίγραφο του ‘config.plist’ για να το επαναφέρεις αργότερα.
  3. Ενημέρωσε στο MMF 3 και δες αν σου αρέσει!

Όταν θέλεις να επιστρέψεις στο MMF 2:

  1. Κατέβασε το MMF 2 ξανά.
  2. Εμφάνισε το ‘config.plist’ στο Finder και αντικατέστησέ το με το αντίγραφο που έκανες νωρίτερα.

Αν το MMF 2 δεν ενεργοποιηθεί αφού επιστρέψεις σε αυτό, μπορεί να χρειαστεί να διαγράψεις το MMF 3, μετά να αδειάσεις τον κάδο απορριμμάτων και μετά να επανεκκινήσεις τον υπολογιστή σου.

Ελπίζω αυτό να σου δίνει ηρεμία όταν δοκιμάζεις το MMF 3.

Νέες λειτουργίες στο Mac Mouse Fix 3!

Το Mac Mouse Fix 3 διαθέτει εκπληκτική νέα κύλιση τύπου trackpad, ένα όμορφο νέο UI, και σου επιτρέπει να κάνεις οτιδήποτε μπορείς να κάνεις σε ένα trackpad εισάγοντας μερικές νέες χειρονομίες.

Δες το σε δράση: Επισκέψου τον νέο ιστότοπο για video demos όλων των λειτουργιών.

Λεπτομερές changelog: Δες τις σημειώσεις έκδοσης για το 3.0.0 Beta 7 και προηγούμενες betas, για να μάθεις όλα όσα άλλαξαν.

Σημαντικότερες νέες λειτουργίες:

  • ‘Κλικ και Σύρσιμο’ σε ‘Κύλιση & Πλοήγηση’

    Μπορείς τώρα να κάνεις κλικ και να σύρεις για να προσομοιώσεις το σύρσιμο με 2 δάχτυλα σε ένα Apple trackpad. Αυτό σου επιτρέπει να περιηγηθείς στο σύστημα με ευκολία καθώς σου επιτρέπει να κάνεις κύλιση πολύ ακριβή και ευέλικτη σε 360°, να πηγαίνεις πίσω και μπροστά στο Safari, να διαγράφεις emails γρήγορα και πολλά άλλα.

    Μπορείς ακόμα να χρησιμοποιήσεις το Swish κατευθείαν από το ποντίκι σου. Είναι μια πολύ ωραία εφαρμογή που είναι σχεδιασμένη να σου επιτρέπει να διαχειρίζεσαι τα παράθυρά σου σύροντας στο trackpad. Αλλά τώρα, μπορείς να τη χρησιμοποιήσεις και με το ποντίκι σου! - Στην πραγματικότητα, οτιδήποτε μπορείς να κάνεις σύροντας σε ένα trackpad λειτουργεί εξίσου καλά με το Mac Mouse Fix!

  • Χειρονομίες ‘Κλικ και Κύλιση’

    Μπορείς τώρα να κρατήσεις πατημένο ένα κουμπί του ποντικιού και μετά να κάνεις κύλιση για να ενεργοποιήσεις πολλές διαφορετικές ενέργειες, όπως εμφάνιση της επιφάνειας εργασίας, μεγέθυνση ή σμίκρυνση και πολλά άλλα. Αισθάνεται εξαιρετικά ανταποκρινόμενο και ρευστό, ακριβώς όπως ένα trackpad.

  • Πλήρως ανακατασκευασμένη κύλιση

    Η κύλιση τώρα απαιτεί λιγότερη προσπάθεια και αισθάνεται πολύ πιο ομαλή και πιο διασκεδαστική στη χρήση. Μπορεί επίσης να προσομοιώσει το σύρσιμο σε ένα trackpad. Αυτό φέρνει μερικές μικρές ευκολίες όπως ένα μικρό αναπήδημα όταν φτάνεις στο τέλος μιας σελίδας, τη δυνατότητα να πηγαίνεις πίσω και μπροστά στο Safari κάνοντας οριζόντια κύλιση, καλύτερη συμβατότητα με το Preview.app και άλλα.

    Επισκέψου την ενότητα κύλισης στον νέο ιστότοπο για να δεις τη νέα κύλιση σε δράση. Αν και νομίζω ότι είναι καλύτερα να τη δοκιμάσεις μόνος σου!

  • Εκπληκτικό νέο UI

    Το UI της εφαρμογής έχει επίσης ανακατασκευαστεί πλήρως για να είναι πιο όμορφο, ισχυρό και διαισθητικό. Μπορεί επίσης να μεταφραστεί σε άλλες γλώσσες τώρα και είναι ήδη διαθέσιμο στα Κινέζικα, Κορεάτικα και Γερμανικά.

  • Στοιχείο γραμμής μενού

    Υπάρχει τώρα ένα στοιχείο γραμμής μενού που μπορείς να ενεργοποιήσεις αν θέλεις, το οποίο σου επιτρέπει να βλέπεις πότε το Mac Mouse Fix είναι ενεργό, και φαίνεται κομψό.

  • Κάνε οτιδήποτε μπορείς να κάνεις σε ένα trackpad

    Αν χρησιμοποιείς ποντίκι 3 κουμπιών, νομίζω ότι η μεγαλύτερη διαφορά μεταξύ MMF 2 και MMF 3 στην καθημερινή σου εμπειρία θα είναι η κύλιση. Αλλά αν χρησιμοποιείς ποντίκι 5 κουμπιών, νομίζω ότι το MMF 3 είναι αρκετά μετασχηματιστικό, γιατί σου επιτρέπει πραγματικά να κάνεις οτιδήποτε μπορείς να κάνεις σε ένα trackpad με απλές και διαισθητικές χειρονομίες στο ποντίκι σου. Για παράδειγμα, Κλικ και Κύλιση για Μεγέθυνση ή σμίκρυνση, Κλικ και Σύρσιμο για να πας Πίσω και Μπροστά στο Safari και πολλά άλλα.

    Επισκέψου την ενότητα trackpad στον νέο ιστότοπο για να δεις αυτές τις νέες χειρονομίες τύπου trackpad σε δράση.

  • Βελτιωμένος μηχανισμός εναλλακτικής λύσης για τιμές που σχετίζονται με την άδεια όταν δεν υπάρχει σύνδεση στο διαδίκτυο
