Mac Mouse Fix 3 を試した後、Mac Mouse Fix 2 に戻す方法

MMF 3 を試して、いつでも MMF 2 に戻すことができます。
ただし、MMF 3 をインストールすると MMF 2 の設定が削除されます。
MMF 3 を試す際にMMF 2 の設定を失わない方法は以下の通りです:

MMF 3 にアップデートする前:

  1. MMF 2 の「More…」画面を開き、キーボードで Command (⌘) と Shift (⇧) を押しながら左上の MMF ロゴをクリックします。Finder に「config.plist」というファイルが表示されます。このファイルにはすべての設定が含まれています。
  2. 後で復元するために「config.plist」のコピーを保存します。
  3. MMF 3 にアップデートして、気に入るか試してみてください!

MMF 2 に戻したい場合:

  1. MMF 2 をダウンロードし直します。
  2. Finder で「config.plist」を表示し、先ほど作成したコピーで置き換えます。

MMF 2 に戻した後に有効にならない場合は、MMF 3 を削除してからゴミ箱を空にし、コンピュータを再起動する必要があるかもしれません。

これで安心して MMF 3 を試せることを願っています。

Mac Mouse Fix 3 の新機能!

Mac Mouse Fix 3 は、素晴らしい新しいトラックパッドのようなスクロール、美しい新しい UIを搭載し、いくつかの新しいジェスチャーを導入することでトラックパッドでできることすべてを可能にします。

実際の動作を見る: 新しいウェブサイトですべての機能のビデオデモをご覧ください。

詳細な変更履歴: 3.0.0 Beta 7 および以前のベータ版のリリースノートを参照して、すべての変更内容を確認してください。

最も重要な新機能:

  • 「クリック&ドラッグ」で「スクロール&ナビゲート」

    クリック&ドラッグで Apple トラックパッドの2本指スワイプをシミュレートできるようになりました。これにより、360°で非常に正確かつ柔軟にスクロールしたり、Safari で前後に移動したり、メールを素早く削除したりなど、システムをスムーズに操作できます。

    マウスから直接 Swish を使うこともできます。これはトラックパッドでスワイプしてウィンドウを管理できるように設計された素晴らしいアプリです。でも今は、マウスでも使えます! - 実際、トラックパッドでスワイプしてできることは、Mac Mouse Fix でも同じようにできます!

  • 「クリック&スクロール」ジェスチャー

    マウスボタンを押したままスクロールすることで、デスクトップの表示、ズームイン/アウトなど、さまざまなアクションをトリガーできるようになりました。トラックパッドのように超反応的で滑らかに感じられます。

  • 完全に再構築されたスクロール

    スクロールがより少ない労力で済み、はるかに滑らかで楽しく使えるようになりました。トラックパッドでのスワイプもシミュレートできます。これにより、ページの端に到達したときの小さなバウンス、Safari で水平スクロールして前後に移動できること、Preview.app との互換性の向上など、細かな改善がもたらされます。

    新しいウェブサイトのスクロールセクションで新しいスクロールの動作をご覧ください。ただし、実際に試してみるのが一番だと思います!

  • 素晴らしい新しい UI

    アプリの UI も完全に再構築され、より美しく、パワフルで、直感的になりました。他の言語にも翻訳できるようになり、すでに中国語、韓国語、ドイツ語で利用可能です。

  • メニューバー項目

    お好みで有効にできるメニューバー項目が追加され、Mac Mouse Fix がアクティブなときに確認でき、スタイリッシュな見た目です。

  • トラックパッドでできることすべて

    3ボタンマウスを使用している場合、MMF 2 と MMF 3 の日常的な体験の最大の違いはスクロールだと思います。しかし、5ボタンマウスを使用している場合、MMF 3 は非常に革新的だと思います。なぜなら、マウスでのシンプルで直感的なジェスチャーで、トラックパッドでできることすべてを本当に実現できるからです。例えば、クリック&スクロールでズームイン/アウト、クリック&ドラッグで Safari で前後に移動など、さまざまなことができます。

    新しいウェブサイトのトラックパッドセクションで、これらの新しいトラックパッドのようなジェスチャーの動作をご覧ください。
