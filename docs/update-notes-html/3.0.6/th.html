ℹ️ แปลโดย AI

บันทึกการเปิดตัวนี้ได้รับการแปลโดย Claude AI อาจมีข้อผิดพลาดบางประการ
สำหรับเวอร์ชันภาษาอังกฤษต้นฉบับ คลิกที่นี่

Mac Mouse Fix 3.0.6 ทำให้ฟีเจอร์ ‘ย้อนกลับ’ และ ‘ไปข้างหน้า’ รองรับแอปได้มากขึ้น นอกจากนี้ยังแก้ไขบั๊กและปัญหาต่างๆ อีกหลายอย่าง

ปรับปรุงฟีเจอร์ ‘ย้อนกลับ’ และ ‘ไปข้างหน้า’

การกำหนดปุ่มเมาส์ ‘ย้อนกลับ’ และ ‘ไปข้างหน้า’ ตอนนี้ใช้งานได้ในแอปมากขึ้น รวมถึง:

  • Visual Studio Code, Cursor, VSCodium, Windsurf, Zed และโปรแกรมแก้ไขโค้ดอื่นๆ
  • แอปในตัวของ Apple หลายตัว เช่น Preview, Notes, System Settings, App Store และ Music
  • Adobe Acrobat
  • Zotero
  • และอื่นๆ อีกมากมาย!

การพัฒนาได้รับแรงบันดาลใจจากฟีเจอร์ ‘Universal Back and Forward’ ที่ยอดเยี่ยมใน LinearMouse ซึ่งควรรองรับแอปทั้งหมดที่ LinearMouse รองรับ
นอกจากนี้ยังรองรับบางแอปที่ปกติต้องใช้แป้นพิมพ์ลัดเพื่อย้อนกลับและไปข้างหน้า เช่น System Settings, App Store, Apple Notes และ Adobe Acrobat โดย Mac Mouse Fix จะตรวจจับแอปเหล่านั้นและจำลองแป้นพิมพ์ลัดที่เหมาะสม

ทุกแอปที่เคยถูกขอใน GitHub Issue ควรได้รับการรองรับแล้ว! (ขอบคุณสำหรับความคิดเห็น!)
หากคุณพบแอปใดที่ยังใช้งานไม่ได้ โปรดแจ้งให้ฉันทราบใน feature request

แก้ไขบั๊ก ‘การเลื่อนหยุดทำงานเป็นครั้งคราว’

ผู้ใช้บางคนประสบปัญหาที่การเลื่อนแบบนุ่มนวลหยุดทำงานแบบสุ่ม

แม้ว่าฉันจะไม่เคยสามารถทำให้เกิดปัญหานี้ซ้ำได้ แต่ฉันได้พัฒนาวิธีแก้ไขที่เป็นไปได้:

แอปจะลองใหม่หลายครั้งเมื่อการตั้งค่าการซิงโครไนซ์กับจอแสดงผลล้มเหลว
หากยังใช้งานไม่ได้หลังจากลองใหม่ แอปจะ:

  • รีสตาร์ทกระบวนการพื้นหลัง ‘Mac Mouse Fix Helper’ ซึ่งอาจแก้ไขปัญหาได้
  • สร้างรายงานข้อขัดข้อง ซึ่งอาจช่วยวินิจฉัยบั๊กได้

ฉันหวังว่าปัญหาจะได้รับการแก้ไขแล้ว! หากยังไม่ได้รับการแก้ไข โปรดแจ้งให้ฉันทราบใน bug report หรือทาง email

ปรับปรุงพฤติกรรมของล้อเลื่อนแบบหมุนอิสระ

Mac Mouse Fix จะไม่เพิ่มความเร็วในการเลื่อนให้คุณอีกต่อไป เมื่อคุณปล่อยให้ล้อเลื่อนหมุนอิสระบนเมาส์ MX Master (หรือเมาส์อื่นๆ ที่มีล้อเลื่อนแบบหมุนอิสระ)

แม้ว่าฟีเจอร์ ‘เพิ่มความเร็วการเลื่อน’ นี้จะมีประโยชน์บนล้อเลื่อนทั่วไป แต่บนล้อเลื่อนแบบหมุนอิสระอาจทำให้ควบคุมได้ยากขึ้น

หมายเหตุ: Mac Mouse Fix ปัจจุบันยังไม่รองรับเมาส์ Logitech ส่วนใหญ่อย่างเต็มที่ รวมถึง MX Master ฉันวางแผนที่จะเพิ่มการรองรับอย่างเต็มรูปแบบ แต่อาจใช้เวลาสักพัก ในระหว่างนี้ ไดรเวอร์บุคคลที่สามที่ดีที่สุดที่ฉันรู้จักซึ่งรองรับ Logitech คือ SteerMouse

การแก้ไขบั๊ก

  • แก้ไขปัญหาที่ Mac Mouse Fix บางครั้งจะเปิดใช้งานแป้นพิมพ์ลัดที่เคยถูกปิดใช้งานใน System Settings อีกครั้ง
  • แก้ไขข้อขัดข้องเมื่อคลิก ‘Activate License’
  • แก้ไขข้อขัดข้องเมื่อคลิก ‘Cancel’ ทันทีหลังจากคลิก ‘Activate License’ (ขอบคุณสำหรับรายงาน Ali!)
  • แก้ไขข้อขัดข้องเมื่อพยายามใช้ Mac Mouse Fix ขณะที่ไม่มีจอแสดงผลเชื่อมต่อกับ Mac ของคุณ
  • แก้ไข memory leak และปัญหาภายในอื่นๆ เมื่อสลับระหว่างแท็บในแอป

การปรับปรุงด้านภาพ

  • แก้ไขปัญหาที่แท็บ About บางครั้งสูงเกินไป ซึ่งเกิดขึ้นใน 3.0.5
  • ข้อความบนการแจ้งเตือน ‘Free days are over’ ไม่ถูกตัดออกอีกต่อไปในภาษาจีน
  • แก้ไขปัญหาภาพที่เงาของฟิลด์ ‘+’ หลังจากบันทึกอินพุต
  • แก้ไขปัญหาที่หายากที่ข้อความตัวอย่างบนหน้าจอ ‘Enter Your License Key’ จะปรากฏไม่อยู่ตรงกลาง
  • แก้ไขปัญหาที่สัญลักษณ์บางตัวที่แสดงในแอปมีสีผิดหลังจากสลับระหว่างโหมดมืด/สว่าง

การปรับปรุงอื่นๆ

  • ทำให้แอนิเมชันบางอย่าง เช่น แอนิเมชันการสลับแท็บ มีประสิทธิภาพมากขึ้นเล็กน้อย
  • ปิดการใช้งานการเติมข้อความอัตโนมัติของ Touch Bar บนหน้าจอ ‘Enter Your License Key’
  • การปรับปรุงภายในเล็กๆ น้อยๆ อื่นๆ อีกหลายอย่าง

แก้ไขด้วยความช่วยเหลืออันยอดเยี่ยมจาก Claude
