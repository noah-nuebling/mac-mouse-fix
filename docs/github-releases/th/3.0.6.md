# 3.0.6
***วันที่เผยแพร่:** 11 ส.ค. 2025*

<br>

Mac Mouse Fix **3.0.6** ทำให้ฟีเจอร์ 'ย้อนกลับ' และ 'ไปข้างหน้า' รองรับแอปได้มากขึ้น
นอกจากนี้ยังแก้ไขบั๊กและปัญหาต่างๆ อีกหลายอย่าง

### ปรับปรุงฟีเจอร์ 'ย้อนกลับ' และ 'ไปข้างหน้า'

การกำหนดปุ่มเมาส์ 'ย้อนกลับ' และ 'ไปข้างหน้า' ตอนนี้**ใช้งานได้ในแอปมากขึ้น** รวมถึง:

- Visual Studio Code, Cursor, VSCodium, Windsurf, Zed และโปรแกรมแก้ไขโค้ดอื่นๆ
- แอปในตัวของ Apple หลายตัว เช่น Preview, Notes, System Settings, App Store และ Music
- Adobe Acrobat
- Zotero
- และอื่นๆ อีกมากมาย!

การพัฒนาได้รับแรงบันดาลใจจากฟีเจอร์ 'Universal Back and Forward' ที่ยอดเยี่ยมใน [LinearMouse](https://github.com/linearmouse/linearmouse) ซึ่งควรรองรับแอปทั้งหมดที่ LinearMouse รองรับ \
นอกจากนี้ยังรองรับบางแอปที่ปกติต้องใช้แป้นพิมพ์ลัดเพื่อย้อนกลับและไปข้างหน้า เช่น System Settings, App Store, Apple Notes และ Adobe Acrobat โดย Mac Mouse Fix จะตรวจจับแอปเหล่านั้นและจำลองแป้นพิมพ์ลัดที่เหมาะสม

ทุกแอปที่เคย[ถูกขอใน GitHub Issue](https://github.com/noah-nuebling/mac-mouse-fix/issues?q=state%3Aclosed%20label%3A%22Universal%20Back%20and%20Forward%22) ควรได้รับการรองรับแล้ว! (ขอบคุณสำหรับความคิดเห็น!) \
หากคุณพบแอปใดที่ยังใช้งานไม่ได้ โปรดแจ้งให้ฉันทราบใน [feature request](http://redirect.macmousefix.com/?target=mmf-feedback-feature-request&locale=th)



### แก้ไขบั๊ก 'การเลื่อนหยุดทำงานเป็นครั้งคราว'

ผู้ใช้บางคนประสบ[ปัญหา](https://github.com/noah-nuebling/mac-mouse-fix/issues?q=is%3Aissue%20state%3Aclosed%20stops%20working%20label%3A%22Scroll%20Stops%20Working%20Intermittently%22)ที่**การเลื่อนแบบนุ่มนวลหยุดทำงาน**แบบสุ่ม

แม้ว่าฉันจะไม่เคยสามารถทำให้เกิดปัญหานี้ซ้ำได้ แต่ฉันได้พัฒนาวิธีแก้ไขที่เป็นไปได้:

แอปจะลองใหม่หลายครั้งเมื่อการตั้งค่าการซิงโครไนซ์กับจอแสดงผลล้มเหลว \
หากยังใช้งานไม่ได้หลังจากลองใหม่ แอปจะ:

- รีสตาร์ทกระบวนการพื้นหลัง 'Mac Mouse Fix Helper' ซึ่งอาจแก้ไขปัญหาได้
- สร้างรายงานข้อขัดข้อง ซึ่งอาจช่วยวินิจฉัยบั๊กได้

ฉันหวังว่าปัญหาจะได้รับการแก้ไขแล้ว! หากยังไม่ได้รับการแก้ไข โปรดแจ้งให้ฉันทราบใน [bug report](http://redirect.macmousefix.com/?target=mmf-feedback-bug-report&locale=th) หรือทาง [email](http://redirect.macmousefix.com/?target=mailto-noah&locale=th)



### ปรับปรุงพฤติกรรมของล้อเลื่อนแบบหมุนอิสระ

Mac Mouse Fix จะ**ไม่เพิ่มความเร็วในการเลื่อน**ให้คุณอีกต่อไป เมื่อคุณปล่อยให้ล้อเลื่อนหมุนอิสระบนเมาส์ MX Master (หรือเมาส์อื่นๆ ที่มีล้อเลื่อนแบบหมุนอิสระ)

แม้ว่าฟีเจอร์ 'เพิ่มความเร็วการเลื่อน' นี้จะมีประโยชน์บนล้อเลื่อนทั่วไป แต่บนล้อเลื่อนแบบหมุนอิสระอาจทำให้ควบคุมได้ยากขึ้น

**หมายเหตุ:** Mac Mouse Fix ปัจจุบันยังไม่รองรับเมาส์ Logitech ส่วนใหญ่อย่างเต็มที่ รวมถึง MX Master ฉันวางแผนที่จะเพิ่มการรองรับอย่างเต็มรูปแบบ แต่อาจใช้เวลาสักพัก ในระหว่างนี้ ไดรเวอร์บุคคลที่สามที่ดีที่สุดที่ฉันรู้จักซึ่งรองรับ Logitech คือ [SteerMouse](https://plentycom.jp/en/steermouse/)





### การแก้ไขบั๊ก

- แก้ไขปัญหาที่ Mac Mouse Fix บางครั้งจะเปิดใช้งานแป้นพิมพ์ลัดที่เคยถูกปิดใช้งานใน System Settings อีกครั้ง
- แก้ไขข้อขัดข้องเมื่อคลิก 'Activate License'
- แก้ไขข้อขัดข้องเมื่อคลิก 'Cancel' ทันทีหลังจากคลิก 'Activate License' (ขอบคุณสำหรับรายงาน Ali!)
- แก้ไขข้อขัดข้องเมื่อพยายามใช้ Mac Mouse Fix ขณะที่ไม่มีจอแสดงผลเชื่อมต่อกับ Mac ของคุณ
- แก้ไข memory leak และปัญหาภายในอื่นๆ เมื่อสลับระหว่างแท็บในแอป

### การปรับปรุงด้านภาพ

- แก้ไขปัญหาที่แท็บ About บางครั้งสูงเกินไป ซึ่งเกิดขึ้นใน [3.0.5](https://redirect.macmousefix.com/?target=mmf-release&tag=3.0.5&locale=th)
- ข้อความบนการแจ้งเตือน 'Free days are over' ไม่ถูกตัดออกอีกต่อไปในภาษาจีน
- แก้ไขปัญหาภาพที่เงาของฟิลด์ '+' หลังจากบันทึกอินพุต
- แก้ไขปัญหาที่หายากที่ข้อความตัวอย่างบนหน้าจอ 'Enter Your License Key' จะปรากฏไม่อยู่ตรงกลาง
- แก้ไขปัญหาที่สัญลักษณ์บางตัวที่แสดงในแอปมีสีผิดหลังจากสลับระหว่างโหมดมืด/สว่าง

### การปรับปรุงอื่นๆ

- ทำให้แอนิเมชันบางอย่าง เช่น แอนิเมชันการสลับแท็บ มีประสิทธิภาพมากขึ้นเล็กน้อย
- ปิดการใช้งานการเติมข้อความอัตโนมัติของ Touch Bar บนหน้าจอ 'Enter Your License Key'
- การปรับปรุงภายในเล็กๆ น้อยๆ อื่นๆ อีกหลายอย่าง
