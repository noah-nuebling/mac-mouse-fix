# 3.0.5
***リリース日:** 2025/07/16*

<br>

Mac Mouse Fix **3.0.5** では、いくつかのバグを修正し、パフォーマンスを改善し、アプリに少し磨きをかけました。\
また、macOS 26 Tahoe にも対応しています。

### トラックパッドスクロールのシミュレーション改善

- スクロールシステムが、トラックパッドでの2本指タップをシミュレートして、アプリケーションのスクロールを停止できるようになりました。
    - これにより、iPhoneやiPadアプリを実行している際に、ユーザーが停止を選択した後もスクロールが続いてしまう問題が修正されました。
- トラックパッドから指を離すシミュレーションの不整合を修正しました。
    - これにより、一部の状況で最適でない動作が発生していた可能性があります。



### macOS 26 Tahoe 互換性

macOS 26 Tahoe ベータを実行している場合、アプリが使用可能になり、UIのほとんどが正しく動作するようになりました。



### パフォーマンス向上

クリック&ドラッグで「スクロール&ナビゲート」ジェスチャーのパフォーマンスを改善しました。\
私のテストでは、CPU使用率が約50%削減されました!

**背景**

「スクロール&ナビゲート」ジェスチャー中、Mac Mouse Fix は透明なウィンドウに偽のマウスカーソルを描画し、実際のマウスカーソルを固定します。これにより、マウスをどれだけ動かしても、スクロールを開始したUI要素をスクロールし続けることができます。

パフォーマンスの改善は、この透明なウィンドウでデフォルトのmacOSイベント処理をオフにすることで実現されました。これは使用されていませんでした。





### バグ修正

- Wacomペンタブレットからのスクロールイベントを無視するようになりました。
    - 以前は、Mac Mouse Fix がWacomタブレットで不規則なスクロールを引き起こしていました。これはGitHub Issue [#1233](https://github.com/noah-nuebling/mac-mouse-fix/issues/1233)で@frenchie1980さんから報告されました。(ありがとうございます!)
    
- Mac Mouse Fix 3.0.4の新しいライセンスシステムの一部として導入されたSwift Concurrencyコードが、正しいスレッドで実行されていなかったバグを修正しました。
    - これにより、macOS Tahoeでクラッシュが発生し、ライセンス周りで他の散発的なバグも発生していた可能性があります。
- オフラインライセンスをデコードするコードの堅牢性を改善しました。
    - これにより、AppleのAPIの問題を回避できます。この問題により、私のIntel Mac Miniでオフラインライセンスの検証が常に失敗していました。これはすべてのIntel Macで発生していたと思われ、GitHub Issue [#1356](https://github.com/noah-nuebling/mac-mouse-fix/issues/1356)で@toni20k5267さんから報告されたように、3.0.4で既に対処された「無料期間が終了しました」バグが一部の人に発生し続けた理由だと考えられます。(ありがとうございます!)
        - 「無料期間が終了しました」バグが発生した方には、申し訳ございませんでした!返金は[こちら](https://redirect.macmousefix.com/?target=mmf-apply-for-refund&locale=ja)から申請できます。
     
     

### UX改善

- Mac Mouse Fixを有効にできないmacOSのバグに対する段階的な解決策を提供するダイアログを無効にしました。
    - これらの問題はmacOS 13 VenturaとmacOS 14 Sonomaでのみ発生していました。現在、これらのダイアログは関連するmacOSバージョンでのみ表示されます。
    - また、ダイアログのトリガーも少し厳しくなりました。以前は、あまり役に立たない状況で表示されることがありました。
    
- 「無料期間が終了しました」通知に「ライセンスを有効化」リンクを直接追加しました。
    - これにより、Mac Mouse Fixライセンスの有効化がさらに手間いらずになります!

### ビジュアル改善

- 「ソフトウェアアップデート」ウィンドウの外観を少し改善しました。macOS 26 Tahoeにより適合するようになりました。
    - これは、Mac Mouse Fixがアップデートを処理するために使用している「Sparkle 1.27.3」フレームワークのデフォルトの外観をカスタマイズすることで実現されました。
- 中国語で「情報」タブの下部のテキストが切れることがある問題を、ウィンドウを少し広くすることで修正しました。
- 「情報」タブの下部のテキストが少しずれていた問題を修正しました。
- 「ボタン」タブの「キーボードショートカット...」オプションの下のスペースが小さすぎるバグを修正しました。

### 内部的な変更

- 「SnapKit」フレームワークへの依存を削除しました。
    - これにより、アプリのサイズが19.8 MBから19.5 MBにわずかに削減されました。
- コードベースのその他の小さな改善。

*Claudeの優れた支援により編集されました。*
