# 3.0.3
***リリース日:** 2024/09/19*

<br>

Mac Mouse Fix **3.0.3** は macOS 15 Sequoia に対応しました。また、いくつかの安定性の問題を修正し、複数の小さな改善を提供します。

### macOS 15 Sequoia サポート

このアプリは macOS 15 Sequoia で正常に動作するようになりました!

- macOS 15 Sequoia では、ほとんどの UI アニメーションが壊れていました。今ではすべてが再び正常に動作しています!
- ソースコードが macOS 15 Sequoia でビルド可能になりました。以前は、Swift コンパイラの問題によりアプリのビルドができませんでした。

### スクロールクラッシュへの対処

Mac Mouse Fix 3.0.2 以降、スクロール中に Mac Mouse Fix が定期的に無効化され、再び有効化されるという[複数の報告](https://github.com/noah-nuebling/mac-mouse-fix/issues/988)がありました。これは「Mac Mouse Fix Helper」バックグラウンドアプリのクラッシュが原因でした。このアップデートでは、以下の変更によりこれらのクラッシュの修正を試みています:

- スクロール機構は、これらのクラッシュを引き起こしたと思われるエッジケースに遭遇した場合、クラッシュする代わりに回復して動作を継続しようとします。
- アプリ全体で予期しない状態の処理方法を変更しました:常に即座にクラッシュする代わりに、多くの場合、アプリは予期しない状態から回復しようとするようになりました。
    
    - この変更は、上記のスクロールクラッシュの修正に貢献しています。他のクラッシュも防ぐ可能性があります。

補足:私は自分のマシンでこれらのクラッシュを再現することができず、何が原因だったのかまだ確信が持てませんが、受け取った報告に基づくと、このアップデートでクラッシュを防げるはずです。もしスクロール中にまだクラッシュが発生する場合、または 3.0.2 でクラッシュを*経験した*場合は、GitHub Issue [#988](https://github.com/noah-nuebling/mac-mouse-fix/issues/988) であなたの経験と診断データを共有していただけると貴重です。これにより、問題を理解し、Mac Mouse Fix を改善することができます。ありがとうございます!

### スクロールのカクつきへの対処

3.0.2 では、Apple の VSync API の問題によって引き起こされる可能性のあるスクロールのカクつきを減らすために、Mac Mouse Fix がシステムにスクロールイベントを送信する方法を変更しました。

しかし、より広範なテストとフィードバックの後、3.0.2 の新しい機構は一部のシナリオではスクロールをよりスムーズにしますが、他のシナリオではよりカクつくことがわかりました。特に Firefox では明らかに悪化しているようでした。\
全体的に、新しい機構が実際にスクロールのカクつきを全面的に改善したかどうかは明確ではありませんでした。また、上記のスクロールクラッシュに寄与した可能性もあります。

そのため、新しい機構を無効にし、スクロールイベントの VSync 機構を Mac Mouse Fix 3.0.0 および 3.0.1 の状態に戻しました。

詳細については、GitHub Issue [#875](https://github.com/noah-nuebling/mac-mouse-fix/issues/875) を参照してください。

### 返金

3.0.1 および 3.0.2 のスクロール変更に関連するトラブルについてお詫び申し上げます。私はそれに伴う問題を大幅に過小評価しており、これらの問題への対処が遅れました。この経験から学び、今後はこのような変更により慎重になるよう最善を尽くします。また、影響を受けた方には返金を提供したいと思います。ご希望の方は[こちら](https://redirect.macmousefix.com/?target=mmf-apply-for-refund&locale=ja)をクリックしてください。

### よりスマートなアップデート機構

これらの変更は Mac Mouse Fix [2.2.4](https://redirect.macmousefix.com/?target=mmf-release&tag=2.2.4&locale=ja) および [2.2.5](https://redirect.macmousefix.com/?target=mmf-release&tag=2.2.5&locale=ja) から引き継がれました。詳細については、それらのリリースノートをご覧ください。概要は以下の通りです:

- ユーザーに表示するアップデートを決定する、新しくよりスマートな機構が追加されました。
- Sparkle 1.26.0 アップデートフレームワークから最新の Sparkle [1.27.3](https://github.com/sparkle-project/Sparkle/releases/tag/1.27.3) に切り替えました。
- Mac Mouse Fix の新しいバージョンが利用可能であることを通知するためにアプリが表示するウィンドウが JavaScript をサポートするようになり、アップデートノートのより良いフォーマットが可能になりました。

### その他の改善とバグ修正

- 「About」タブでアプリの価格と関連情報が場合によって正しく表示されない問題を修正しました。
- 複数のディスプレイを使用している間、スムーズスクロールをディスプレイのリフレッシュレートと同期する機構が正常に動作しない問題を修正しました。
- 多数の細かい内部的なクリーンアップと改善。
