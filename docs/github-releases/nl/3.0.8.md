# 3.0.8
***Publicatiedatum:** 12 sep 2025*

<br>

Mac Mouse Fix **3.0.8** lost UI-problemen en meer op.

### **UI-problemen**

- Het nieuwe ontwerp is uitgeschakeld op macOS 26 Tahoe. Nu ziet de app eruit en werkt deze zoals op macOS 15 Sequoia. 
    - Ik heb dit gedaan omdat sommige van Apple's vernieuwde UI-elementen nog steeds problemen hebben. Bijvoorbeeld, de '-' knoppen op het 'Knoppen' tabblad waren niet altijd klikbaar.
    - De UI kan er nu een beetje gedateerd uitzien op macOS 26 Tahoe. Maar deze zou volledig functioneel en gepolijst moeten zijn, net als voorheen.
- Een bug opgelost waarbij de 'Gratis dagen zijn voorbij' melding vast kwam te zitten in de rechterbovenhoek van het scherm.
    - Dank aan [Sashpuri](https://github.com/Sashpuri) en anderen voor het melden!

### **UI-verfijning**

- De groene verkeerslichtknop in het hoofdvenster van Mac Mouse Fix is uitgeschakeld.
    - De knop deed niets, omdat het venster niet handmatig kan worden vergroot of verkleind.
- Een probleem opgelost waarbij sommige horizontale lijnen in de tabel op het 'Knoppen' tabblad te donker waren onder macOS 26 Tahoe.
- Een bug opgelost waarbij het bericht "Primaire muisknop kan niet worden gebruikt" op het 'Knoppen' tabblad soms werd afgesneden onder macOS 26 Tahoe.
- Een typfout in de Duitse interface gecorrigeerd. Met dank aan GitHub-gebruiker [i-am-the-slime](https://github.com/i-am-the-slime). Bedankt!
- Een probleem opgelost waarbij het MMF-venster soms kort flitste in de verkeerde grootte bij het openen van het venster op macOS 26 Tahoe.

### **Andere wijzigingen**

- Verbeterd gedrag bij het proberen Mac Mouse Fix in te schakelen terwijl meerdere instanties van Mac Mouse Fix op de computer draaien. 
    - Mac Mouse Fix zal nu ijveriger proberen de andere instantie van Mac Mouse Fix uit te schakelen. 
    - Dit kan randgevallen verbeteren waarbij Mac Mouse Fix niet kon worden ingeschakeld.
- Onder-de-motorkap wijzigingen en opruiming.
