# 3.0.3
***Ημερομηνία κυκλοφορίας:** 19 Σεπ 2024*

<br>

Το Mac Mouse Fix **3.0.3** είναι έτοιμο για το macOS 15 Sequoia. Επίσης διορθώνει ορισμένα προβλήματα σταθερότητας και παρέχει αρκετές μικρές βελτιώσεις.

### Υποστήριξη macOS 15 Sequoia

Η εφαρμογή τώρα λειτουργεί σωστά στο macOS 15 Sequoia!

- Οι περισσότερες κινούμενες εικόνες του UI δεν λειτουργούσαν στο macOS 15 Sequoia. Τώρα όλα λειτουργούν σωστά ξανά!
- Ο πηγαίος κώδικας μπορεί πλέον να μεταγλωττιστεί στο macOS 15 Sequoia. Προηγουμένως, υπήρχαν προβλήματα με τον μεταγλωττιστή Swift που εμπόδιζαν τη μεταγλώττιση της εφαρμογής.

### Αντιμετώπιση κατάρρευσης κύλισης

Από το Mac Mouse Fix 3.0.2 υπήρξαν [πολλαπλές αναφορές](https://github.com/noah-nuebling/mac-mouse-fix/issues/988) ότι το Mac Mouse Fix απενεργοποιείται και επανενεργοποιείται περιοδικά κατά την κύλιση. Αυτό προκαλούνταν από καταρρεύσεις της εφαρμογής παρασκηνίου 'Mac Mouse Fix Helper'. Αυτή η ενημέρωση προσπαθεί να διορθώσει αυτές τις καταρρεύσεις, με τις ακόλουθες αλλαγές:

- Ο μηχανισμός κύλισης θα προσπαθήσει να ανακάμψει και να συνεχίσει να λειτουργεί αντί να καταρρεύσει, όταν συναντά την ακραία περίπτωση που φαίνεται να οδήγησε σε αυτές τις καταρρεύσεις.
- Άλλαξα τον τρόπο με τον οποίο χειρίζονται οι απροσδόκητες καταστάσεις στην εφαρμογή πιο γενικά: Αντί να καταρρέει πάντα αμέσως, η εφαρμογή θα προσπαθεί τώρα να ανακάμψει από απροσδόκητες καταστάσεις σε πολλές περιπτώσεις.
    
    - Αυτή η αλλαγή συμβάλλει στις διορθώσεις για τις καταρρεύσεις κύλισης που περιγράφονται παραπάνω. Μπορεί επίσης να αποτρέψει άλλες καταρρεύσεις.
  
Σημείωση: Δεν μπόρεσα ποτέ να αναπαράγω αυτές τις καταρρεύσεις στο μηχάνημά μου, και ακόμα δεν είμαι σίγουρος τι τις προκάλεσε, αλλά με βάση τις αναφορές που έλαβα, αυτή η ενημέρωση θα πρέπει να αποτρέψει οποιεσδήποτε καταρρεύσεις. Αν εξακολουθείς να αντιμετωπίζεις καταρρεύσεις κατά την κύλιση ή αν *αντιμετώπισες* καταρρεύσεις στην έκδοση 3.0.2, θα ήταν πολύτιμο αν μοιραζόσουν την εμπειρία σου και τα διαγνωστικά δεδομένα στο GitHub Issue [#988](https://github.com/noah-nuebling/mac-mouse-fix/issues/988). Αυτό θα με βοηθούσε να κατανοήσω το πρόβλημα και να βελτιώσω το Mac Mouse Fix. Ευχαριστώ!

### Αντιμετώπιση τραντάγματος κύλισης

Στην έκδοση 3.0.2 έκανα αλλαγές στον τρόπο με τον οποίο το Mac Mouse Fix στέλνει γεγονότα κύλισης στο σύστημα σε μια προσπάθεια να μειώσω τα τραντάγματα κύλισης που πιθανώς προκαλούνται από προβλήματα με τα VSync APIs της Apple.

Ωστόσο, μετά από πιο εκτεταμένες δοκιμές και σχόλια, φαίνεται ότι ο νέος μηχανισμός στην έκδοση 3.0.2 κάνει την κύλιση πιο ομαλή σε ορισμένα σενάρια αλλά πιο τραντάζουσα σε άλλα. Ειδικά στον Firefox φαινόταν να είναι αισθητά χειρότερη. \
Συνολικά, δεν ήταν σαφές ότι ο νέος μηχανισμός πραγματικά βελτίωσε τα τραντάγματα κύλισης σε όλες τις περιπτώσεις. Επίσης, μπορεί να συνέβαλε στις καταρρεύσεις κύλισης που περιγράφονται παραπάνω.

Γι' αυτό απενεργοποίησα τον νέο μηχανισμό και επανέφερα τον μηχανισμό VSync για τα γεγονότα κύλισης πίσω στον τρόπο που ήταν στο Mac Mouse Fix 3.0.0 και 3.0.1.

Δες το GitHub Issue [#875](https://github.com/noah-nuebling/mac-mouse-fix/issues/875) για περισσότερες πληροφορίες.

### Επιστροφή χρημάτων

Λυπάμαι για τα προβλήματα που σχετίζονται με τις αλλαγές κύλισης στις εκδόσεις 3.0.1 και 3.0.2. Υποτίμησα σημαντικά τα προβλήματα που θα προέκυπταν από αυτό, και ήμουν αργός στην αντιμετώπιση αυτών των ζητημάτων. Θα κάνω ό,τι μπορώ για να μάθω από αυτή την εμπειρία και να είμαι πιο προσεκτικός με τέτοιες αλλαγές στο μέλλον. Θα ήθελα επίσης να προσφέρω επιστροφή χρημάτων σε όποιον επηρεάστηκε. Απλά κάνε κλικ [εδώ](https://redirect.macmousefix.com/?target=mmf-apply-for-refund&locale=el) αν σε ενδιαφέρει.

### Εξυπνότερος μηχανισμός ενημέρωσης

Αυτές οι αλλαγές μεταφέρθηκαν από το Mac Mouse Fix [2.2.4](https://redirect.macmousefix.com/?target=mmf-release&tag=2.2.4&locale=el) και [2.2.5](https://redirect.macmousefix.com/?target=mmf-release&tag=2.2.5&locale=el). Δες τις σημειώσεις έκδοσης τους για να μάθεις περισσότερα για τις λεπτομέρειες. Εδώ είναι μια περίληψη:

- Υπάρχει ένας νέος, εξυπνότερος μηχανισμός που αποφασίζει ποια ενημέρωση θα εμφανιστεί στον χρήστη.
- Αλλαγή από τη χρήση του πλαισίου ενημέρωσης Sparkle 1.26.0 στο πιο πρόσφατο Sparkle [1.27.3](https://github.com/sparkle-project/Sparkle/releases/tag/1.27.3).
- Το παράθυρο που εμφανίζει η εφαρμογή για να σε ενημερώσει ότι είναι διαθέσιμη μια νέα έκδοση του Mac Mouse Fix υποστηρίζει τώρα JavaScript, το οποίο επιτρέπει καλύτερη μορφοποίηση των σημειώσεων ενημέρωσης.

### Άλλες Βελτιώσεις & Διορθώσεις Σφαλμάτων

- Διορθώθηκε ένα πρόβλημα όπου η τιμή της εφαρμογής και οι σχετικές πληροφορίες εμφανίζονταν εσφαλμένα στην καρτέλα 'Σχετικά' σε ορισμένες περιπτώσεις.
- Διορθώθηκε ένα πρόβλημα όπου ο μηχανισμός συγχρονισμού της ομαλής κύλισης με τον ρυθμό ανανέωσης της οθόνης δεν λειτουργούσε σωστά κατά τη χρήση πολλαπλών οθονών.
- Πολλές μικρές βελτιώσεις και καθαρισμός στο παρασκήνιο.
