# 3.0.6
***Ημερομηνία κυκλοφορίας:** 11 Αυγ 2025*

<br>

Το Mac Mouse Fix **3.0.6** κάνει τη λειτουργία 'Πίσω' και 'Μπροστά' συμβατή με περισσότερες εφαρμογές.
Επίσης διορθώνει διάφορα σφάλματα και προβλήματα.

### Βελτιωμένη Λειτουργία 'Πίσω' και 'Μπροστά'

Οι αντιστοιχίσεις των κουμπιών ποντικιού 'Πίσω' και 'Μπροστά' τώρα **λειτουργούν σε περισσότερες εφαρμογές**, συμπεριλαμβανομένων:

- Visual Studio Code, Cursor, VSCodium, Windsurf, Zed και άλλων επεξεργαστών κώδικα
- Πολλές ενσωματωμένες εφαρμογές της Apple όπως το Preview, Notes, System Settings, App Store και Music
- Adobe Acrobat
- Zotero
- Και άλλες!

Η υλοποίηση είναι εμπνευσμένη από την εξαιρετική λειτουργία 'Universal Back and Forward' στο [LinearMouse](https://github.com/linearmouse/linearmouse). Θα πρέπει να υποστηρίζει όλες τις εφαρμογές που υποστηρίζει το LinearMouse. \
Επιπλέον υποστηρίζει κάποιες εφαρμογές που κανονικά απαιτούν συντομεύσεις πληκτρολογίου για να πας πίσω και μπροστά, όπως το System Settings, App Store, Apple Notes και Adobe Acrobat. Το Mac Mouse Fix τώρα θα ανιχνεύει αυτές τις εφαρμογές και θα προσομοιώνει τις κατάλληλες συντομεύσεις πληκτρολογίου.

Κάθε εφαρμογή που έχει ποτέ [ζητηθεί σε GitHub Issue](https://github.com/noah-nuebling/mac-mouse-fix/issues?q=state%3Aclosed%20label%3A%22Universal%20Back%20and%20Forward%22) θα πρέπει να υποστηρίζεται τώρα! (Ευχαριστώ για τα σχόλια!) \
Αν βρεις εφαρμογές που δεν λειτουργούν ακόμα, ενημέρωσέ με σε ένα [αίτημα λειτουργίας](http://redirect.macmousefix.com/?target=mmf-feedback-feature-request&locale=el).



### Αντιμετώπιση του Σφάλματος 'Η Κύλιση Σταματά να Λειτουργεί Περιστασιακά'

Κάποιοι χρήστες αντιμετώπισαν ένα [πρόβλημα](https://github.com/noah-nuebling/mac-mouse-fix/issues?q=is%3Aissue%20state%3Aclosed%20stops%20working%20label%3A%22Scroll%20Stops%20Working%20Intermittently%22) όπου **η ομαλή κύλιση σταματά να λειτουργεί** τυχαία.

Ενώ δεν μπόρεσα ποτέ να αναπαράγω το πρόβλημα, έχω υλοποιήσει μια πιθανή διόρθωση:

Η εφαρμογή τώρα θα επαναπροσπαθεί πολλές φορές, όταν αποτυγχάνει η ρύθμιση του συγχρονισμού οθόνης. \
Αν εξακολουθεί να μην λειτουργεί μετά τις επαναπροσπάθειες, η εφαρμογή θα:

- Επανεκκινήσει τη διεργασία παρασκηνίου 'Mac Mouse Fix Helper', που μπορεί να λύσει το πρόβλημα
- Παράγει μια αναφορά σφάλματος, που μπορεί να βοηθήσει στη διάγνωση του σφάλματος

Ελπίζω το πρόβλημα να έχει λυθεί τώρα! Αν όχι, ενημέρωσέ με σε μια [αναφορά σφάλματος](http://redirect.macmousefix.com/?target=mmf-feedback-bug-report&locale=el) ή μέσω [email](http://redirect.macmousefix.com/?target=mailto-noah&locale=el).



### Βελτιωμένη Συμπεριφορά Ροδέλας Κύλισης με Ελεύθερη Περιστροφή

Το Mac Mouse Fix **δεν θα επιταχύνει πλέον την κύλιση** για σένα, όταν αφήνεις τη ροδέλα κύλισης να περιστρέφεται ελεύθερα στο ποντίκι MX Master. (Ή σε οποιοδήποτε άλλο ποντίκι με ροδέλα κύλισης ελεύθερης περιστροφής.)

Ενώ αυτή η λειτουργία 'επιτάχυνσης κύλισης' είναι χρήσιμη σε κανονικές ροδέλες κύλισης, σε μια ροδέλα ελεύθερης περιστροφής μπορεί να κάνει τα πράγματα πιο δύσκολα στον έλεγχο.

**Σημείωση:** Το Mac Mouse Fix προς το παρόν δεν είναι πλήρως συμβατό με τα περισσότερα ποντίκια Logitech, συμπεριλαμβανομένου του MX Master. Σχεδιάζω να προσθέσω πλήρη υποστήριξη, αλλά πιθανόν να πάρει λίγο χρόνο. Στο μεταξύ, το καλύτερο πρόγραμμα οδήγησης τρίτου κατασκευαστή με υποστήριξη Logitech που γνωρίζω είναι το [SteerMouse](https://plentycom.jp/en/steermouse/).





### Διορθώσεις Σφαλμάτων

- Διορθώθηκε ένα πρόβλημα όπου το Mac Mouse Fix μερικές φορές επανενεργοποιούσε συντομεύσεις πληκτρολογίου που είχαν απενεργοποιηθεί προηγουμένως στις Ρυθμίσεις Συστήματος  
- Διορθώθηκε ένα σφάλμα κατά το κλικ στο 'Ενεργοποίηση Άδειας' 
- Διορθώθηκε ένα σφάλμα κατά το κλικ στο 'Ακύρωση' αμέσως μετά το κλικ στο 'Ενεργοποίηση Άδειας' (Ευχαριστώ για την αναφορά, Ali!)
- Διορθώθηκαν σφάλματα κατά την προσπάθεια χρήσης του Mac Mouse Fix όταν δεν είναι συνδεδεμένη οθόνη στο Mac σου 
- Διορθώθηκε μια διαρροή μνήμης και κάποια άλλα εσωτερικά προβλήματα κατά την εναλλαγή μεταξύ καρτελών στην εφαρμογή 

### Οπτικές Βελτιώσεις

- Διορθώθηκε ένα πρόβλημα όπου η καρτέλα About ήταν μερικές φορές πολύ ψηλή, που εισήχθη στην [3.0.5](https://redirect.macmousefix.com/?target=mmf-release&tag=3.0.5&locale=el)
- Το κείμενο στην ειδοποίηση 'Free days are over' δεν κόβεται πλέον στα Κινέζικα
- Διορθώθηκε ένα οπτικό πρόβλημα στη σκιά του πεδίου '+' μετά την εγγραφή μιας εισόδου
- Διορθώθηκε ένα σπάνιο πρόβλημα όπου το κείμενο υπόδειξης στην οθόνη 'Enter Your License Key' εμφανιζόταν εκτός κέντρου
- Διορθώθηκε ένα πρόβλημα όπου κάποια σύμβολα που εμφανίζονταν στην εφαρμογή είχαν λάθος χρώμα μετά την εναλλαγή μεταξύ σκοτεινής/φωτεινής λειτουργίας

### Άλλες Βελτιώσεις

- Έγιναν κάποια κινούμενα γραφικά, όπως το κινούμενο γραφικό εναλλαγής καρτελών, ελαφρώς πιο αποδοτικά  
- Απενεργοποιήθηκε η αυτόματη συμπλήρωση κειμένου Touch Bar στην οθόνη 'Enter Your License Key' 
- Διάφορες μικρότερες εσωτερικές βελτιώσεις
