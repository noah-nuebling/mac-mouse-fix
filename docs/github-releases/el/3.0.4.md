# 3.0.4
***Ημερομηνία κυκλοφορίας:** 7 Ιουν 2025*

<br>

Το Mac Mouse Fix **3.0.4** βελτιώνει το απόρρητο, την αποδοτικότητα και την αξιοπιστία.\
Εισάγει ένα νέο σύστημα αδειοδότησης εκτός σύνδεσης και διορθώνει αρκετά σημαντικά σφάλματα.

### Ενισχυμένο Απόρρητο & Αποδοτικότητα

Η έκδοση 3.0.4 εισάγει ένα νέο σύστημα επικύρωσης άδειας εκτός σύνδεσης που ελαχιστοποιεί τις συνδέσεις στο διαδίκτυο όσο το δυνατόν περισσότερο.\
Αυτό βελτιώνει το απόρρητο και εξοικονομεί πόρους του συστήματος του υπολογιστή σου.\
Όταν έχεις άδεια χρήσης, η εφαρμογή τώρα λειτουργεί 100% εκτός σύνδεσης!

<details>
<summary><b>Κάνε κλικ εδώ για περισσότερες λεπτομέρειες</b></summary>
Οι προηγούμενες εκδόσεις επικύρωναν τις άδειες online σε κάθε εκκίνηση, επιτρέποντας ενδεχομένως την αποθήκευση αρχείων καταγραφής συνδέσεων από διακομιστές τρίτων (GitHub και Gumroad). Το νέο σύστημα εξαλείφει τις περιττές συνδέσεις – μετά την αρχική ενεργοποίηση της άδειας, συνδέεται στο διαδίκτυο μόνο αν τα τοπικά δεδομένα άδειας είναι κατεστραμμένα.
<br><br>
Ενώ δεν καταγράφηκε ποτέ καμία συμπεριφορά χρήστη από εμένα προσωπικά, το προηγούμενο σύστημα θεωρητικά επέτρεπε σε διακομιστές τρίτων να καταγράφουν διευθύνσεις IP και χρόνους σύνδεσης. Το Gumroad μπορούσε επίσης να καταγράψει το κλειδί άδειας σου και ενδεχομένως να το συσχετίσει με οποιαδήποτε προσωπικά στοιχεία κατέγραψε για εσένα όταν αγόρασες το Mac Mouse Fix.
<br><br>
Δεν είχα λάβει υπόψη αυτά τα λεπτά ζητήματα απορρήτου όταν δημιούργησα το αρχικό σύστημα αδειοδότησης, αλλά τώρα, το Mac Mouse Fix είναι όσο το δυνατόν πιο ιδιωτικό και χωρίς διαδίκτυο!
<br><br>
Δες επίσης την <a href=https://gumroad.com/privacy>πολιτική απορρήτου του Gumroad</a> και αυτό το <a href=https://github.com/noah-nuebling/mac-mouse-fix/issues/976#issuecomment-2140955801>σχόλιο στο GitHub</a> από εμένα.

</details>

### Διορθώσεις Σφαλμάτων

- Διορθώθηκε ένα σφάλμα όπου το macOS μερικές φορές κολλούσε όταν χρησιμοποιούσες 'Click and Drag' για 'Spaces & Mission Control'.
- Διορθώθηκε ένα σφάλμα όπου οι συντομεύσεις πληκτρολογίου στις Ρυθμίσεις Συστήματος μερικές φορές διαγράφονταν όταν χρησιμοποιούσες ενέργειες 'Click' του Mac Mouse Fix όπως το 'Mission Control'.
- Διορθώθηκε [ένα σφάλμα](https://github.com/noah-nuebling/mac-mouse-fix/issues?q=state%3Aopen%20label%3A%22%27Free%20days%20are%20over%27%20bug%22) όπου η εφαρμογή μερικές φορές σταματούσε να λειτουργεί και εμφάνιζε μια ειδοποίηση ότι οι 'Δωρεάν ημέρες τελείωσαν' σε χρήστες που είχαν ήδη αγοράσει την εφαρμογή.
    - Αν αντιμετώπισες αυτό το σφάλμα, ζητώ ειλικρινά συγγνώμη για την ταλαιπωρία. Μπορείς να υποβάλεις αίτηση για [επιστροφή χρημάτων εδώ](https://redirect.macmousefix.com/?message=&target=mmf-apply-for-refund&locale=el).
- Βελτιώθηκε ο τρόπος με τον οποίο η εφαρμογή ανακτά το κύριο παράθυρό της, κάτι που μπορεί να διόρθωσε ένα σφάλμα όπου η οθόνη 'Ενεργοποίηση Άδειας' μερικές φορές αποτυγχάνει να εμφανιστεί.

### Βελτιώσεις Ευχρηστίας

- Έγινε αδύνατη η εισαγωγή κενών και αλλαγών γραμμής στο πεδίο κειμένου της οθόνης 'Ενεργοποίηση Άδειας'.
    - Αυτό ήταν ένα συνηθισμένο σημείο σύγχυσης, επειδή είναι πολύ εύκολο να επιλέξεις κατά λάθος μια κρυφή αλλαγή γραμμής όταν αντιγράφεις το κλειδί άδειας σου από τα emails του Gumroad.
- Αυτές οι σημειώσεις ενημέρωσης μεταφράζονται αυτόματα για μη αγγλόφωνους χρήστες (Με την υποστήριξη του Claude). Ελπίζω να είναι χρήσιμο! Αν αντιμετωπίσεις οποιοδήποτε πρόβλημα, ενημέρωσέ με. Αυτή είναι μια πρώτη ματιά σε ένα νέο σύστημα μετάφρασης που αναπτύσσω τον τελευταίο χρόνο.

### Διακοπή (Ανεπίσημης) Υποστήριξης για macOS 10.14 Mojave

Το Mac Mouse Fix 3 υποστηρίζει επίσημα macOS 11 Big Sur και νεότερα. Ωστόσο, για χρήστες πρόθυμους να αποδεχτούν κάποια προβλήματα και γραφικά ζητήματα, το Mac Mouse Fix 3.0.3 και παλαιότερα μπορούσαν ακόμα να χρησιμοποιηθούν σε macOS 10.14.4 Mojave.

Το Mac Mouse Fix 3.0.4 διακόπτει αυτή την υποστήριξη και **τώρα απαιτεί macOS 10.15 Catalina**. \
Ζητώ συγγνώμη για οποιαδήποτε ταλαιπωρία προκαλεί αυτό. Αυτή η αλλαγή μου επέτρεψε να υλοποιήσω το βελτιωμένο σύστημα αδειοδότησης χρησιμοποιώντας σύγχρονες λειτουργίες Swift. Οι χρήστες Mojave μπορούν να συνεχίσουν να χρησιμοποιούν το Mac Mouse Fix [3.0.3](https://redirect.macmousefix.com/?target=mmf-release&tag=3.0.3&locale=el) ή την [τελευταία έκδοση του Mac Mouse Fix 2](https://redirect.macmousefix.com/?target=mmf2-latest&locale=el). Ελπίζω ότι αυτή είναι μια καλή λύση για όλους.

### Βελτιώσεις Στο Παρασκήνιο

- Υλοποιήθηκε ένα νέο σύστημα 'MFDataClass' που επιτρέπει πιο ισχυρή μοντελοποίηση δεδομένων διατηρώντας το αρχείο διαμόρφωσης του Mac Mouse Fix αναγνώσιμο και επεξεργάσιμο από ανθρώπους.
- Δημιουργήθηκε υποστήριξη για προσθήκη πλατφορμών πληρωμών εκτός από το Gumroad. Έτσι στο μέλλον, μπορεί να υπάρχουν τοπικοποιημένες ολοκληρώσεις αγορών και η εφαρμογή θα μπορούσε να πωληθεί σε διαφορετικές χώρες.
- Βελτιωμένη καταγραφή που μου επιτρέπει να δημιουργώ πιο αποτελεσματικές "Debug Builds" για χρήστες που αντιμετωπίζουν δύσκολα στην αναπαραγωγή σφάλματα.
- Πολλές άλλες μικρές βελτιώσεις και εργασίες καθαρισμού.

*Επεξεργασμένο με εξαιρετική βοήθεια από το Claude.*
