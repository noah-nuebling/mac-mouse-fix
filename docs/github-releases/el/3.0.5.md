# 3.0.5
***Ημερομηνία κυκλοφορίας:** 16 Ιουλ 2025*

<br>

Το Mac Mouse Fix **3.0.5** διορθώνει αρκετά σφάλματα, βελτιώνει την απόδοση και προσθέτει λίγη στίλβωση στην εφαρμογή. \
Είναι επίσης συμβατό με το macOS 26 Tahoe.

### Βελτιωμένη Προσομοίωση Κύλισης Trackpad

- Το σύστημα κύλισης μπορεί τώρα να προσομοιώσει ένα χτύπημα με δύο δάχτυλα στο trackpad για να σταματήσουν οι εφαρμογές να κάνουν κύλιση.
    - Αυτό διορθώνει ένα πρόβλημα κατά την εκτέλεση εφαρμογών iPhone ή iPad, όπου η κύλιση συχνά συνέχιζε αφού ο χρήστης επέλεγε να σταματήσει.
- Διορθώθηκε η ασυνεπής προσομοίωση του σηκώματος των δαχτύλων από το trackpad.
    - Αυτό μπορεί να προκαλούσε μη βέλτιστη συμπεριφορά σε ορισμένες καταστάσεις.



### Συμβατότητα με macOS 26 Tahoe

Όταν εκτελείται το macOS 26 Tahoe Beta, η εφαρμογή είναι τώρα χρησιμοποιήσιμη και το μεγαλύτερο μέρος του UI λειτουργεί σωστά.



### Βελτίωση Απόδοσης

Βελτιωμένη απόδοση της χειρονομίας Κλικ και Σύρσιμο για "Κύλιση & Πλοήγηση". \
Στις δοκιμές μου, η χρήση CPU έχει μειωθεί κατά ~50%!

**Υπόβαθρο**

Κατά τη διάρκεια της χειρονομίας "Κύλιση & Πλοήγηση", το Mac Mouse Fix σχεδιάζει έναν ψεύτικο δρομέα ποντικιού σε ένα διαφανές παράθυρο, ενώ κλειδώνει τον πραγματικό δρομέα ποντικιού στη θέση του. Αυτό διασφαλίζει ότι μπορείς να συνεχίσεις να κάνεις κύλιση στο στοιχείο UI που ξεκίνησες να κάνεις κύλιση, ανεξάρτητα από το πόσο μακριά μετακινείς το ποντίκι σου.

Η βελτιωμένη απόδοση επιτεύχθηκε απενεργοποιώντας τον προεπιλεγμένο χειρισμό συμβάντων του macOS σε αυτό το διαφανές παράθυρο, ο οποίος ούτως ή άλλως δεν χρησιμοποιούνταν.





### Διορθώσεις Σφαλμάτων

- Τώρα αγνοούνται τα συμβάντα κύλισης από τις ταμπλέτες σχεδίασης Wacom.
    - Πριν, το Mac Mouse Fix προκαλούσε ασταθή κύλιση στις ταμπλέτες Wacom, όπως αναφέρθηκε από τον @frenchie1980 στο GitHub Issue [#1233](https://github.com/noah-nuebling/mac-mouse-fix/issues/1233). (Ευχαριστούμε!)
    
- Διορθώθηκε ένα σφάλμα όπου ο κώδικας Swift Concurrency, που εισήχθη ως μέρος του νέου συστήματος αδειοδότησης στο Mac Mouse Fix 3.0.4, δεν εκτελούνταν στο σωστό thread.
    - Αυτό προκαλούσε crashes στο macOS Tahoe, και πιθανότατα προκαλούσε και άλλα σποραδικά σφάλματα σχετικά με την αδειοδότηση.
- Βελτιωμένη ανθεκτικότητα του κώδικα που αποκωδικοποιεί τις offline άδειες.
    - Αυτό παρακάμπτει ένα πρόβλημα στα APIs της Apple που προκαλούσε την επικύρωση offline άδειας να αποτυγχάνει πάντα στο Intel Mac Mini μου. Υποθέτω ότι αυτό συνέβαινε σε όλα τα Intel Mac, και ότι ήταν ο λόγος για τον οποίο το σφάλμα "Free days are over" (που είχε ήδη αντιμετωπιστεί στο 3.0.4) εξακολουθούσε να εμφανίζεται για μερικούς ανθρώπους, όπως αναφέρθηκε από τον @toni20k5267 στο GitHub Issue [#1356](https://github.com/noah-nuebling/mac-mouse-fix/issues/1356). (Ευχαριστούμε!)
        - Αν αντιμετώπισες το σφάλμα "Free days are over", λυπάμαι γι' αυτό! Μπορείς να πάρεις επιστροφή χρημάτων [εδώ](https://redirect.macmousefix.com/?target=mmf-apply-for-refund&locale=el).
     
     

### Βελτιώσεις UX

- Απενεργοποιήθηκαν οι διάλογοι που παρείχαν βήμα προς βήμα λύσεις για σφάλματα του macOS που εμπόδιζαν τους χρήστες να ενεργοποιήσουν το Mac Mouse Fix.
    - Αυτά τα προβλήματα εμφανίζονταν μόνο στο macOS 13 Ventura και 14 Sonoma. Τώρα, αυτοί οι διάλογοι εμφανίζονται μόνο σε εκείνες τις εκδόσεις macOS όπου είναι σχετικοί. 
    - Οι διάλογοι είναι επίσης λίγο πιο δύσκολο να ενεργοποιηθούν – πριν, μερικές φορές εμφανίζονταν σε καταστάσεις όπου δεν ήταν πολύ χρήσιμοι.
    
- Προστέθηκε ένας σύνδεσμος "Ενεργοποίηση Άδειας" απευθείας στην ειδοποίηση "Free days are over". 
    - Αυτό κάνει την ενεργοποίηση μιας άδειας Mac Mouse Fix ακόμα πιο εύκολη!

### Οπτικές Βελτιώσεις

- Βελτιώθηκε ελαφρώς η εμφάνιση του παραθύρου "Ενημέρωση Λογισμικού". Τώρα ταιριάζει καλύτερα με το macOS 26 Tahoe. 
    - Αυτό έγινε προσαρμόζοντας την προεπιλεγμένη εμφάνιση του framework "Sparkle 1.27.3" που χρησιμοποιεί το Mac Mouse Fix για τη διαχείριση ενημερώσεων.
- Διορθώθηκε το πρόβλημα όπου το κείμενο στο κάτω μέρος της καρτέλας Σχετικά μερικές φορές κόβονταν στα Κινέζικα, κάνοντας το παράθυρο ελαφρώς πιο φαρδύ.
- Διορθώθηκε το κείμενο στο κάτω μέρος της καρτέλας Σχετικά που ήταν ελαφρώς εκτός κέντρου.
- Διορθώθηκε ένα σφάλμα που προκαλούσε ο χώρος κάτω από την επιλογή "Συντόμευση Πληκτρολογίου..." στην καρτέλα Κουμπιά να είναι πολύ μικρός. 

### Αλλαγές Κάτω από την Κουκούλα

- Αφαιρέθηκε η εξάρτηση από το framework "SnapKit".
    - Αυτό μειώνει ελαφρώς το μέγεθος της εφαρμογής από 19.8 σε 19.5 MB.
- Διάφορες άλλες μικρές βελτιώσεις στον κώδικα.

*Επεξεργασμένο με εξαιρετική βοήθεια από τον Claude.*
